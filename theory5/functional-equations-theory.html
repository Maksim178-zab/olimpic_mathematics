📐 Функциональные уравнения

Полная теория для олимпиадников

📖 Часть 1. Что такое функциональное уравнение?

1.1. Определение

Функциональное уравнение — это уравнение, связывающее значения неизвестной функции в разных точках.

Примеры:

  • \( f(x+y) = f(x) + f(y) \) — уравнение Коши (линейная функция).
  • \( f(xy) = f(x)f(y) \) — мультипликативное уравнение.
  • \( f(x+y) = f(x)f(y) \) — показательная функция.
  • \( f(x+y) + f(x-y) = 2f(x)f(y) \) — уравнение для косинуса.

1.2. Почему это сложно?

В обычном уравнении мы ищем число. В функциональном — функцию, то есть бесконечное множество значений. Решений может быть много, и нужно доказать, что мы нашли все.

«Функциональное уравнение решается тогда, когда ты перепробовал все возможные подстановки, и одна из них сработала».

🔧 Часть 2. Базовые методы решения

2.1. Подстановка конкретных значений

Суть: подставляем вместо переменных удобные числа (0, 1, -1) или выражаем одну переменную через другую.

Пример: Найти все \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие:

\[ f(x+y) = f(x) + f(y). \]

Решение:

Шаг 1. Подставляем \( x = y = 0 \):

\[ f(0) = f(0) + f(0) \quad\Rightarrow\quad f(0) = 0. \]

Шаг 2. Подставляем \( y = -x \):

\[ f(0) = f(x) + f(-x) \quad\Rightarrow\quad 0 = f(x) + f(-x) \quad\Rightarrow\quad f(-x) = -f(x). \]

Функция нечётная.

Шаг 3. Для целых \( n \) легко показать по индукции: \( f(n) = n f(1) \).

Шаг 4. Для рациональных \( \frac{p}{q} \):

\[ f(p) = f\left(q \cdot \frac{p}{q}\right) = q f\left(\frac{p}{q}\right). \]

Значит, \( f\left(\frac{p}{q}\right) = \frac{p}{q} f(1) \).

Шаг 5. Для всех действительных чисел нужна дополнительная гладкость (непрерывность, монотонность, ограниченность). Если её нет, существуют дикие решения (через базис Гамеля).

Шаг 6. Если предположить непрерывность, то \( f(x) = cx \).

Вывод: подстановка нуля и единицы — всегда первый шаг.

2.2. Симметризация (замена переменных)

Суть: делаем замену \( x \to y \), \( y \to x \) и сравниваем.

Пример: Найти все \( f: \mathbb{R} \to \mathbb{R} \):

\[ f(x+y) = f(x) + f(y) + xy. \]

Решение:

Шаг 1. Подставляем \( x = y = 0 \):

\[ f(0) = f(0) + f(0) + 0 \quad\Rightarrow\quad f(0) = 0. \]

Шаг 2. Подставляем \( y = -x \):

\[ f(0) = f(x) + f(-x) - x^2 \quad\Rightarrow\quad f(x) + f(-x) = x^2. \]

Шаг 3. Подставляем \( y = 1 \):

\[ f(x+1) = f(x) + f(1) + x. \]

Это уже даёт рекуррентное соотношение.

Шаг 4. Пробуем \( f(x) = \frac{x^2}{2} + cx \). Проверяем:

\[ f(x+y) = \frac{(x+y)^2}{2} + c(x+y) = \frac{x^2}{2} + xy + \frac{y^2}{2} + cx + cy. \] \[ f(x) + f(y) + xy = \frac{x^2}{2} + cx + \frac{y^2}{2} + cy + xy. \]

Совпадает. Значит, это решение.

2.3. Подстановка выражений

Суть: выбираем замены, чтобы упростить уравнение или получить новые соотношения.

Пример: \( f(x+y) = f(x)f(y) \).

Шаг 1. \( y = 0 \): \( f(x) = f(x)f(0) \).

Если \( f(0) \neq 1 \), то \( f(x) = 0 \) для всех \( x \).

Шаг 2. Если \( f(0) = 1 \), то \( f(0) = f(x)f(-x) \) ⇒ \( f(-x) = \frac{1}{f(x)} \).

Шаг 3. По индукции \( f(n) = f(1)^n \).

Шаг 4. При непрерывности: \( f(x) = a^x \).

2.4. Доказательство монотонности или ограниченности

Суть: после подстановок мы часто получаем уравнение Коши. Чтобы перейти от рациональных чисел ко всем действительным, нужно доказать, что функция непрерывна, монотонна или ограничена на интервале.

Пример: если \( f(x+y) = f(x) + f(y) \) и \( f \) монотонна, то \( f(x) = cx \).

Доказательство:

Шаг 1. Для рациональных \( q \): \( f(q) = q f(1) \).

Шаг 2. Для любого \( x \) возьмём две рациональные последовательности \( q_n \to x \) слева и \( r_n \to x \) справа.

Шаг 3. Из монотонности: \( q_n f(1) = f(q_n) \le f(x) \le f(r_n) = r_n f(1) \).

Шаг 4. Переходя к пределу, получаем \( f(x) = x f(1) \).

⚡ Часть 3. Специальные методы

3.1. Метод «подставь \( y = x \)»

Иногда подстановка \( y = x \) даёт уравнение, которое легко решить.

Пример: \( f(x+y) = f(x) + f(y) + 2xy \).

Подставим \( y = x \):

\[ f(2x) = 2f(x) + 2x^2. \]

Это уже не уравнение Коши, а рекуррентное соотношение. Можно искать решение в виде \( f(x) = x^2 + cx \).

3.2. Метод «подставь \( y = \frac{1}{x} \)»

Для уравнений, содержащих \( f(x) \) и \( f(1/x) \), часто помогает подстановка \( y = 1/x \).

Пример: \( f(x) + f\left(\frac{1}{x}\right) = x \).

Подставим \( x \to \frac{1}{x} \):

\[ f\left(\frac{1}{x}\right) + f(x) = \frac{1}{x}. \]

Сравниваем с исходным:

\[ x = \frac{1}{x} \quad\Rightarrow\quad x^2 = 1. \]

Это даёт противоречие для всех \( x \), кроме \( \pm 1 \). Значит, решений нет (или только в частных точках).

3.3. Метод конечных разностей

Суть: подставляем \( x + y \) и \( x - y \), затем вычитаем или складываем.

Пример: \( f(x+y) + f(x-y) = 2f(x)f(y) \).

Шаг 1. Подставим \( y = 0 \):

\[ f(x) + f(x) = 2f(x)f(0) \quad\Rightarrow\quad f(x) = f(x)f(0). \]

Если \( f(x) \neq 0 \), то \( f(0) = 1 \).

Шаг 2. Подставим \( x = 0 \):

\[ f(y) + f(-y) = 2f(0)f(y) = 2f(y). \]

Значит, \( f(-y) = f(y) \) — функция чётная.

Шаг 3. Это уравнение для косинуса: \( \cos(x+y) + \cos(x-y) = 2\cos x \cos y \).

Шаг 4. Решения: \( f(x) = 0 \), \( f(x) = \cos(cx) \), \( f(x) = \cosh(cx) \).

3.4. Метод замены \( g(x) = f(x) - f(0) \)

Иногда удобно сдвинуть функцию, чтобы избавиться от свободного члена.

Пример: \( f(x+y) = f(x) + f(y) + 1 \).

Попробуем \( g(x) = f(x) + 1 \). Тогда:

\[ g(x+y) - 1 = (g(x)-1) + (g(y)-1) + 1 = g(x) + g(y) - 1. \] \[ g(x+y) = g(x) + g(y). \]

Это уже уравнение Коши. Решение: \( g(x) = cx \), значит \( f(x) = cx - 1 \).

📚 Часть 4. Классические типы функциональных уравнений

4.1. Уравнение Коши

\[ f(x+y) = f(x) + f(y). \]

Решение (при дополнительных условиях): \( f(x) = cx \).

4.2. Уравнение Йенсена

\[ f\left(\frac{x+y}{2}\right) = \frac{f(x) + f(y)}{2}. \]

Решение (при непрерывности): \( f(x) = ax + b \).

4.3. Уравнение Даламбера

\[ f(x+y) + f(x-y) = 2f(x)f(y). \]

Решение: \( f(x) = 0 \), \( f(x) = \cos(cx) \), \( f(x) = \cosh(cx) \).

4.4. Уравнение для экспоненты

\[ f(x+y) = f(x)f(y). \]

Решение: \( f(x) = 0 \) или \( f(x) = a^x \).

4.5. Уравнение для логарифма

\[ f(xy) = f(x) + f(y). \]

Решение: \( f(x) = c \ln x \) (на \( \mathbb{R}^+ \)).

🏆 Часть 5. Полный разбор олимпиадной задачи

Задача (IMO 2017, вариант):

Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие:

\[ f(x+f(y)) = f(x) + y \quad \forall x, y \in \mathbb{R}. \]

Решение (пошагово):

Шаг 1. Подставим \( x = 0 \):

\[ f(f(y)) = f(0) + y. \]

Значит, \( f(f(y)) = y + c \), где \( c = f(0) \).

Шаг 2. Покажем, что \( f \) — биекция.

Если \( f(a) = f(b) \), то из \( f(f(a)) = a + c \) и \( f(f(b)) = b + c \) следует \( a = b \). Значит, инъективна.

Для любого \( t \in \mathbb{R} \) выберем \( y = t - c \), тогда \( f(f(y)) = t \), значит, сюръективна.

Шаг 3. Используем инъективность.

В исходном уравнении заменим \( y \) на \( f(y) \):

\[ f(x + f(f(y))) = f(x) + f(y). \]

Но \( f(f(y)) = y + c \). Значит:

\[ f(x + y + c) = f(x) + f(y). \]

Шаг 4. Сделаем замену \( g(x) = f(x) - c \).

Тогда:

\[ g(x+y+c) + c = (g(x)+c) + (g(y)+c) = g(x) + g(y) + 2c. \] \[ g(x+y+c) = g(x) + g(y) + c. \]

Подставим \( y = 0 \):

\[ g(x+c) = g(x) + g(0) + c. \]

Отсюда можно показать, что \( g \) — линейная: \( g(x) = kx + b \).

Шаг 5. Подставляем обратно и находим константы.

После подстановки получаем: \( f(x) = x + c \) или \( f(x) = -x + c \).

Ответ: \( f(x) = x + a \) или \( f(x) = -x + a \), где \( a \in \mathbb{R} \).

📋 Часть 6. Шпаргалка для олимпиадника

Ситуация Метод
Есть \( f(0) \) Подставить \( x = 0 \) или \( y = 0 \)
Нужно найти \( f(-x) \) Подставить \( y = -x \)
Есть симметрия \( f(x,y) \) и \( f(y,x) \) Поменять переменные и сравнить
Есть \( f(x) \) и \( f(1/x) \) Подставить \( x \to 1/x \)
Получилось уравнение Коши Доказать монотонность/непрерывность
Есть \( f(x+y) \) и \( f(x-y) \) Вычесть или сложить два уравнения
Есть произвольная константа Сделать замену \( g(x) = f(x) - f(0) \)
Нужно доказать биективность Использовать инъективность и сюръективность

🎯 Заключение

Функциональные уравнения — это искусство. Здесь нет универсального рецепта, но есть набор приёмов, которые нужно тренировать. Главное — не бояться подставлять, сравнивать и искать закономерности.

«Функциональное уравнение решается тогда, когда ты перепробовал все возможные подстановки, и одна из них сработала». Так что пробуйте, ошибайтесь, ищите — и успех придёт.
