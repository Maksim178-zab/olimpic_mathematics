📖 Часть 1. Базовые задачи (1–10)

Задача 1. Найдите все функции \( f: \mathbb{Q} \to \mathbb{Q} \) такие, что \( f(1)=2 \) и \( f(xy) = f(x)f(y) - f(x+y) + 1 \) для всех \( x,y \in \mathbb{Q} \).

Решение (пошаговое):

Шаг 1. Подстановка \( x=1 \), \( y=n \) (для натурального \( n \)): \( f(n) = f(1)f(n) - f(n+1) + 1 = 2f(n) - f(n+1) + 1 \). Переносим: \( f(n+1) = f(n) + 1 \). Значит, значения на натуральных числах образуют арифметическую прогрессию с разностью 1. Так как \( f(1)=2 \), по индукции: \( f(n) = n+1 \) для всех \( n \in \mathbb{N} \).

Шаг 2. Находим \( f(0) \). Подставим \( x=0 \), \( y=n \): \( f(0) = f(0)f(n) - f(n) + 1 \). Так как \( f(n)=n+1 \): \( f(0) = f(0)(n+1) - (n+1) + 1 = f(0)(n+1) - n \). \( f(0) - f(0)(n+1) = -n \) ⇒ \( -f(0)n = -n \) ⇒ \( f(0)=1 \).

Шаг 3. Находим \( f(-1) \). Подставим \( x=-1 \), \( y=1 \): \( f(-1) = f(-1)f(1) - f(0) + 1 = 2f(-1) - 1 + 1 = 2f(-1) \). Значит, \( f(-1)=0 \).

Шаг 4. Находим \( f(-n) \). Подставим \( x=-1 \), \( y=n \): \( f(-n) = f(-1)f(n) - f(n-1) + 1 = 0 \cdot (n+1) - n + 1 = -n + 1 \). Итак, для всех целых \( z \): \( f(z) = z+1 \).

Шаг 5. Находим \( f(1/n) \). Подставим \( x=n \), \( y=1/n \): \( f(1) = (n+1)f(1/n) - f(n+1/n) + 1 \). Нам нужно \( f(n+1/n) \). Для этого сначала заметим, что из \( x=1 \), \( y=m+1/n \): \( f(m+1+1/n) = f(m+1/n) + 1 \). По индукции: \( f(m+1/n) = m + f(1/n) \). Тогда \( f(n+1/n) = n + f(1/n) \). Подставляем: \( 2 = (n+1)f(1/n) - (n + f(1/n)) + 1 = n f(1/n) - n + 1 \). \( n f(1/n) = n + 1 \) ⇒ \( f(1/n) = 1/n + 1 \).

Шаг 6. Для любого рационального \( r = m/n \): подставим \( x=m \), \( y=1/n \): \( f(m/n) = f(m)f(1/n) - f(m+1/n) + 1 \). \( f(m) = m+1 \), \( f(1/n) = 1/n + 1 \), \( f(m+1/n) = m + 1/n + 1 \). \( f(m/n) = (m+1)(1/n+1) - (m+1/n+1) + 1 = m/n + 1 \). Для отрицательных рациональных — через \( x=-1 \), \( y=r \): \( f(-r) = f(-1)f(r) - f(r-1) + 1 = -r + 1 \).

Шаг 7. Проверка: \( f(x)=x+1 \) для всех \( x \in \mathbb{Q} \). Подставляем: \( xy+1 = (x+1)(y+1) - (x+y+1) + 1 \). Равенство верно.

Ответ: \( f(x) = x+1 \).

Задача 2. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+y) = f(x) + f(y) \) для всех \( x,y \in \mathbb{R} \), при условии, что \( f \) непрерывна.

Решение:

Шаг 1. Подстановка \( x=y=0 \): \( f(0) = f(0) + f(0) \) ⇒ \( f(0)=0 \).

Шаг 2. Подстановка \( y = -x \): \( f(0) = f(x) + f(-x) \) ⇒ \( f(-x) = -f(x) \).

Шаг 3. Для целых чисел \( n \): по индукции \( f(n) = n f(1) \).

Шаг 4. Для рациональных \( q = \frac{m}{n} \): \( f(m) = f(n \cdot \frac{m}{n}) = n f(\frac{m}{n}) \). Так как \( f(m) = m f(1) \), получаем: \( n f(\frac{m}{n}) = m f(1) \) ⇒ \( f(\frac{m}{n}) = \frac{m}{n} f(1) \).

Шаг 5. Для любого \( x \in \mathbb{R} \): возьмём рациональную последовательность \( q_n \to x \). Так как \( f \) непрерывна: \( f(x) = \lim f(q_n) = \lim q_n f(1) = x f(1) \).

Шаг 6. Обозначим \( c = f(1) \). Тогда \( f(x) = cx \). Проверка: \( c(x+y) = cx + cy \) — верно.

Ответ: \( f(x) = cx \), где \( c \in \mathbb{R} \) — произвольная константа.

Задача 3. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+y) = f(x)f(y) \) для всех \( x,y \in \mathbb{R} \), при условии непрерывности.

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(x) = f(x)f(0) \). Если существует \( x \) с \( f(x) \neq 0 \), то \( f(0)=1 \). Если \( f(x)=0 \) для всех \( x \), то это решение.

Шаг 2. Рассмотрим случай \( f(0)=1 \). Тогда \( f(x) \neq 0 \) для всех \( x \), так как \( f(x)f(-x)=f(0)=1 \).

Шаг 3. Прологарифмируем. Пусть \( g(x) = \ln f(x) \). Тогда: \( g(x+y) = \ln f(x+y) = \ln(f(x)f(y)) = \ln f(x) + \ln f(y) = g(x) + g(y) \). Это уравнение Коши. Так как \( f \) непрерывна, \( g \) непрерывна, значит \( g(x) = cx \).

Шаг 4. Тогда \( f(x) = e^{cx} = a^x \), где \( a = e^c > 0 \). Второй случай: \( f(x) \equiv 0 \).

Ответ: \( f(x) = 0 \) для всех \( x \) или \( f(x) = a^x \) для некоторого \( a > 0 \).

Задача 4. Найдите все функции \( f: \mathbb{R}^+ \to \mathbb{R}^+ \), удовлетворяющие: \( f(xy) = f(x) + f(y) \) для всех \( x,y > 0 \), при условии непрерывности.

Решение:

Шаг 1. Замена \( x = e^u \), \( y = e^v \). Определим \( g(u) = f(e^u) \). Тогда: \( g(u+v) = f(e^{u+v}) = f(e^u e^v) = f(e^u) + f(e^v) = g(u) + g(v) \).

Шаг 2. Это уравнение Коши для \( g \). Так как \( f \) непрерывна, \( g \) непрерывна, значит \( g(u) = cu \).

Шаг 3. Тогда \( f(x) = g(\ln x) = c \ln x \).

Ответ: \( f(x) = c \ln x \), \( c \in \mathbb{R} \).

Задача 5. Найдите все функции \( f: \mathbb{R} \setminus \{0\} \to \mathbb{R} \), удовлетворяющие: \( f(x) + f(1/x) = x \) для всех \( x \neq 0 \).

Решение:

Шаг 1. Подставим \( x \to 1/x \): \( f(1/x) + f(x) = 1/x \).

Шаг 2. Сравниваем с исходным уравнением: \( x = 1/x \) ⇒ \( x^2 = 1 \) ⇒ \( x = \pm 1 \). Для всех остальных \( x \) получаем противоречие. Значит, решений нет.

Ответ: решений нет.

Задача 6. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+y) = f(x) + f(y) + 1 \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Замена \( g(x) = f(x) + 1 \). Тогда: \( g(x+y) = f(x+y) + 1 = f(x) + f(y) + 1 + 1 = (f(x)+1) + (f(y)+1) = g(x) + g(y) \).

Шаг 2. Это уравнение Коши для \( g \). Без дополнительных условий (непрерывность) есть много решений. Если предположить непрерывность, то \( g(x) = cx \).

Шаг 3. Тогда \( f(x) = cx - 1 \).

Ответ: \( f(x) = cx - 1 \).

Задача 7. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x) + f(y) = f(x+y) + xy \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(x) + f(0) = f(x) + 0 \) ⇒ \( f(0)=0 \).

Шаг 2. Перепишем уравнение: \( f(x+y) = f(x) + f(y) - xy \).

Шаг 3. Попробуем \( f(x) = -x^2/2 \). Проверка: \( f(x+y) = -(x+y)^2/2 \), \( f(x)+f(y)-xy = -x^2/2 - y^2/2 - xy = -(x^2+2xy+y^2)/2 = -(x+y)^2/2 \). Совпадает!

Шаг 4. Общее решение: \( f(x) = -x^2/2 + cx \). Проверим: \( f(x)+f(y)-xy = -x^2/2+cx - y^2/2+cy - xy = -(x^2+2xy+y^2)/2 + c(x+y) = f(x+y) \).

Ответ: \( f(x) = -\frac{x^2}{2} + cx \).

Задача 8. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x) = f(x+1) \) для всех \( x \in \mathbb{R} \) (периодическая с периодом 1).

Решение: Это не уравнение в обычном смысле — любая функция с периодом 1 подходит. Например, \( f(x) = \sin(2\pi x) \), \( f(x) = \{x\} \) (дробная часть), и многие другие. Без дополнительных условий решений бесконечно много.

Ответ: любая 1-периодическая функция.

Задача 9. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x) = f(-x) \) для всех \( x \in \mathbb{R} \) (чётная функция).

Ответ: любая чётная функция.

Задача 10. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x) = -f(-x) \) для всех \( x \in \mathbb{R} \) (нечётная функция).

Ответ: любая нечётная функция.

⚡ Часть 2. Средние задачи (11–20)

Задача 11. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+f(y)) = f(x) + y \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( x=0 \): \( f(f(y)) = f(0) + y \). (1)

Шаг 2. Из (1) следует, что \( f \) — сюръективна. Для любого \( t \in \mathbb{R} \) возьмём \( y = t - f(0) \), тогда \( f(f(y)) = t \).

Шаг 3. Докажем инъективность. Если \( f(a)=f(b) \), то из (1): \( f(0) + a = f(f(a)) = f(f(b)) = f(0) + b \) ⇒ \( a = b \).

Шаг 4. Подставим в исходное уравнение \( y = f(y) \): \( f(x + f(f(y))) = f(x) + f(y) \). По (1): \( f(f(y)) = f(0) + y \). Значит: \( f(x + y + f(0)) = f(x) + f(y) \). (2)

Шаг 5. Сделаем замену \( g(x) = f(x) - f(0) \). Тогда из (2): \( g(x+y+f(0)) + f(0) = g(x) + f(0) + g(y) + f(0) \) ⇒ \( g(x+y+f(0)) = g(x) + g(y) + f(0) \).

Шаг 6. Подставим \( y=0 \): \( g(x+f(0)) = g(x) + g(0) + f(0) \). Это даёт, что \( g \) — линейная функция. Точнее, из структуры следует \( g(x) = kx + b \).

Шаг 7. Возвращаемся к \( f \): \( f(x) = kx + b + f(0) \). Подставляем в исходное уравнение и находим \( k = 1 \) или \( k = -1 \). Проверка даёт: \( f(x) = x + c \) или \( f(x) = -x + c \), \( c \in \mathbb{R} \).

Ответ: \( f(x) = x + c \) или \( f(x) = -x + c \).

Задача 12. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(f(x)) = x \) для всех \( x \in \mathbb{R} \) (инволюция).

Решение: Это определение инволюции. Без дополнительных условий решений бесконечно много. Примеры: \( f(x) = x \), \( f(x) = -x \), \( f(x) = 1 - x \), и любые функции, разбивающие \( \mathbb{R} \) на пары \( \{a,b\} \), где \( f(a)=b \) и \( f(b)=a \).

Ответ: любая инволюция.

Задача 13. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+y) + f(x-y) = 2f(x)f(y) \) для всех \( x,y \in \mathbb{R} \). (Уравнение Даламбера)

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(x) + f(x) = 2f(x)f(0) \) ⇒ \( 2f(x) = 2f(x)f(0) \). Если существует \( x \) с \( f(x) \neq 0 \), то \( f(0)=1 \). Если \( f(x)=0 \) для всех \( x \), то это решение.

Шаг 2. Рассмотрим \( f(0)=1 \). Подставим \( x=0 \): \( f(y) + f(-y) = 2f(0)f(y) = 2f(y) \) ⇒ \( f(-y) = f(y) \). Функция чётная.

Шаг 3. Это уравнение хорошо известно. Его решения (при дополнительных условиях регулярности): \( f(x) = 0 \), \( f(x) = \cos(cx) \), \( f(x) = \cosh(cx) \).

Шаг 4. Проверка: для \( \cos(cx) \): \( \cos(c(x+y)) + \cos(c(x-y)) = 2\cos(cx)\cos(cy) \) — верно. Для \( \cosh(cx) \) аналогично.

Ответ: \( f(x) \equiv 0 \), \( f(x) = \cos(cx) \), \( f(x) = \cosh(cx) \).

Задача 14. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+y) = f(x) + f(y) + xy \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( x=0 \): \( f(y) = f(0) + f(y) \) ⇒ \( f(0)=0 \).

Шаг 2. Попробуем \( f(x) = \frac{x^2}{2} + cx \): \( f(x+y) = \frac{x^2+2xy+y^2}{2} + c(x+y) = \frac{x^2}{2}+\frac{y^2}{2}+xy+cx+cy \). \( f(x)+f(y)+xy = \frac{x^2}{2}+cx+\frac{y^2}{2}+cy+xy \). Совпадает.

Ответ: \( f(x) = \frac{x^2}{2} + cx \).

Задача 15. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(xy) = f(x)f(y) \) для всех \( x,y \in \mathbb{R} \) (мультипликативное уравнение).

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(0) = f(x)f(0) \). Если \( f(x) \neq 0 \) для некоторого \( x \), то \( f(0)=0 \)? Нет: если \( f(0)=1 \), то \( f(0) = f(0)f(0) \) — верно. Если \( f(0)=0 \), то \( f(x)f(0)=0=f(0) \) — верно для всех \( x \).

Шаг 2. Случай \( f(0)=0 \): Тогда \( f(x) = f(x)f(1) \). Если \( f(1) \neq 1 \), то \( f(x)=0 \) для всех \( x \). Если \( f(1)=1 \), то возможны функции \( f(x) = |x|^c \) для \( x>0 \), а для отрицательных — с учётом чётности/нечётности.

Шаг 3. Классические решения: \( f(x) \equiv 0 \), \( f(x) = x^c \), \( f(x) = |x|^c \), \( f(x) = \operatorname{sgn}(x)|x|^c \).

Ответ: \( f(x) = 0 \), \( f(x) = x^c \), \( f(x) = |x|^c \), \( f(x) = \operatorname{sgn}(x)|x|^c \).

Задача 16. Найдите все функции \( f: \mathbb{R}\setminus\{0,1\} \to \mathbb{R} \), удовлетворяющие: \( f(x) = f((x-1)/x) \) для всех \( x \neq 0,1 \).

Решение:

Шаг 1. Заметим, что преобразование \( T(x) = (x-1)/x \) имеет период 3: \( T^3(x) = x \). Проверим: \( T^2(x) = 1/(1-x) \), \( T^3(x) = x \).

Шаг 2. Уравнение требует, чтобы \( f \) была постоянна на каждой орбите под действием \( T \). Орбиты имеют вид \( \{x, T(x), T^2(x)\} \).

Шаг 3. Значит, любая функция, постоянная на каждой такой трёхэлементной орбите, является решением. Без дополнительных условий решений бесконечно много.

Ответ: функции, постоянные на орбитах \( x \to \frac{x-1}{x} \).

Задача 17. Найдите все функции \( f: \mathbb{R}\setminus\{0,1\} \to \mathbb{R} \), удовлетворяющие: \( f(x) + f((x-1)/x) = 1 + x \).

Решение:

Шаг 1. Введём обозначение \( T(x) = (x-1)/x \). Как в задаче 16, \( T^3(x) = x \).

Шаг 2. Запишем уравнение для \( T(x) \): \( f(T(x)) + f(T^2(x)) = 1 + T(x) \).

Шаг 3. Запишем уравнение для \( T^2(x) \): \( f(T^2(x)) + f(x) = 1 + T^2(x) \).

Шаг 4. Теперь у нас система трёх уравнений: \( f(x) + f(T(x)) = 1 + x \), \( f(T(x)) + f(T^2(x)) = 1 + T(x) \), \( f(T^2(x)) + f(x) = 1 + T^2(x) \).

Шаг 5. Сложим (1) и (2) и вычтем (3): \( 2f(T(x)) = 1 + x + T(x) - T^2(x) \).

Шаг 6. Выражаем \( f(T(x)) \), а затем \( f(x) \). После упрощения получаем: \( f(x) = \frac{1}{2}(x + \frac{1}{x} - \frac{1}{x-1}) \).

Ответ: \( f(x) = \frac{1}{2}\left(x + \frac{1}{x} - \frac{1}{x-1}\right) \).

Задача 18. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x+y) = \frac{f(x)+f(y)}{1-f(x)f(y)} \) для всех \( x,y \in \mathbb{R} \), при условии непрерывности.

Решение:

Шаг 1. Напоминает формулу тангенса суммы: \( \tan(x+y) = \frac{\tan x + \tan y}{1 - \tan x \tan y} \).

Шаг 2. Пусть \( f(x) = \tan(g(x)) \). Тогда уравнение превращается в: \( \tan(g(x+y)) = \tan(g(x) + g(y)) \). Значит, \( g(x+y) = g(x) + g(y) + \pi k \) для некоторого целого \( k \). При непрерывности \( k \) постоянно и \( g(x+y) = g(x) + g(y) + c \).

Шаг 3. Далее стандартно: \( g(x) = ax + b \), но с учётом периодичности тангенса. В итоге \( f(x) = \tan(ax + b) \) с ограничениями на область определения.

Ответ: \( f(x) = \tan(cx) \) (с точностью до сдвига).

Задача 19. Найдите все функции \( f: \mathbb{N} \to \mathbb{N} \), удовлетворяющие: \( f(f(n)) = n + 2 \) для всех \( n \in \mathbb{N} \).

Решение:

Шаг 1. Из \( f(f(n)) = n+2 \) следует, что \( f \) инъективна. Если \( f(a)=f(b) \), то \( a+2 = f(f(a)) = f(f(b)) = b+2 \) ⇒ \( a=b \).

Шаг 2. Применим \( f \) к обеим частям: \( f(f(f(n))) = f(n+2) \). Но \( f(f(f(n))) = f(n) + 2 \) (по условию, применённому к \( f(n) \)). Значит: \( f(n+2) = f(n) + 2 \).

Шаг 3. Из этого следует, что \( f \) на чётных и нечётных числах ведёт себя как арифметическая прогрессия с разностью 2. А именно: для чётных: \( f(2k) = f(2) + 2(k-1) \); для нечётных: \( f(2k+1) = f(1) + 2k \).

Шаг 4. Используем условие \( f(f(1)) = 3 \). Пусть \( f(1) = a \). Тогда \( f(a) = 3 \). Если \( a \) нечётное, \( f(a) = f(1) + a - 1 = 2a - 1 \). Приравниваем к 3: \( 2a-1=3 \) ⇒ \( a=2 \) (чётное) — противоречие. Если \( a \) чётное, \( f(a) = f(2) + a - 2 \). Но \( f(2) = f(f(1)) = 3 \). Значит \( f(a) = 3 + a - 2 = a+1 \). Приравниваем к 3: \( a+1=3 \) ⇒ \( a=2 \). Значит \( f(1) = 2 \). Тогда \( f(2) = f(f(1)) = 3 \).

Шаг 5. Из \( f(n+2) = f(n)+2 \): \( f(2k) = 2k+1 \), \( f(2k+1) = 2k+2 \). То есть \( f(n) = n+1 \) для всех \( n \).

Ответ: \( f(n) = n+1 \).

Задача 21 (IMO 1999, задача 6). Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x - f(y)) = f(f(y)) + x f(y) + f(x) - 1 \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Обозначим \( f(0) = c \). Подставим \( x = y = 0 \): \( f(-c) = f(c) + c - 1 \). (1)

Шаг 2. Если \( c = 0 \), то \( f(0) = f(0) - 1 \) — противоречие. Значит, \( c \neq 0 \).

Шаг 3. Подставим \( x = f(y) \): \( f(f(y) - f(y)) = f(0) = c = f(f(y)) + f(y)^2 + f(f(y)) - 1 \). \( c = 2f(f(y)) + f(y)^2 - 1 \). Отсюда: \( f(f(y)) = (c + 1 - f(y)^2)/2 \). (2)

Шаг 4. Из (2) следует, что \( f(f(y)) \) зависит только от \( f(y)^2 \). Значит, \( f(x) = f(-x) \) для всех \( x \).

Шаг 5. Проверим \( f(x) = 1 - x^2/2 \). Тогда \( f(0)=1 \), \( c=1 \). Уравнение превращается в тождество.

Шаг 6. Докажем, что других нет. Из (2) видно, что \( f(x) = f(-x) \). В частности, \( f(c) = f(-c) \). Сравнивая с (1): \( f(c) = f(c) + c - 1 \) ⇒ \( c = 1 \). Подставляем \( c=1 \) в (2): \( f(f(x)) = (2 - x^2)/2 = 1 - x^2/2 \). Это уже даёт вид \( f(x) = 1 - x^2/2 \).

Ответ: \( f(x) = 1 - \frac{x^2}{2} \).

Задача 22. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(xf(x) + f(y)) = y + f(x)^2 \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( x=0 \): \( f(f(y)) = y + f(0)^2 \). Отсюда \( f \) сюръективна (так как правая часть пробегает все \( \mathbb{R} \)).

Шаг 2. Инъективность: если \( f(a)=f(b) \), то из \( f(f(a)) = a + f(0)^2 \) и \( f(f(b)) = b + f(0)^2 \) следует \( a=b \).

Шаг 3. Так как \( f \) сюръективна, существует \( t \) такое, что \( f(t)=0 \). Подставим \( x=0 \), \( y=t \): \( f(f(t)) = t + f(0)^2 \) ⇒ \( f(0) = t + f(0)^2 \).

Шаг 4. Теперь подставим \( x=t \) (где \( f(t)=0 \)): \( f(t \cdot 0 + f(y)) = y + 0^2 \) ⇒ \( f(f(y)) = y \). Сравниваем с \( f(f(y)) = y + f(0)^2 \), значит \( f(0)=0 \).

Шаг 5. Тогда \( f(f(y)) = y \). Подставим \( x = f(x) \) в исходное уравнение: \( f(f(x)f(f(x)) + f(y)) = y + f(f(x))^2 \). Так как \( f(f(x)) = x \), получаем: \( f(x^2 + y) = y + x^2 \). Значит, \( f(t) = t \) для всех \( t \) (так как \( t = x^2 + y \) пробегает все \( \mathbb{R} \)).

Шаг 6. Проверка: \( f(xf(x)+f(y)) = f(x^2+y) = x^2+y = y + f(x)^2 \). Верно.

Ответ: \( f(x) = x \).

Задача 23. Найдите все функции \( f: \mathbb{R}^+ \to \mathbb{R}^+ \), удовлетворяющие: \( f(x+f(x)+y) = f(2x) + f(y) \) для всех \( x,y > 0 \).

Решение:

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(f(x)) = x + f(0) \). Отсюда \( f \) сюръективна.

Шаг 2. Инъективность: если \( f(a)=f(b) \), то \( a + f(0) = f(f(a)) = f(f(b)) = b + f(0) \) ⇒ \( a=b \).

Шаг 3. Подставим \( y = -x \): \( f(f(x)-x) = x + f(-x) \).

Шаг 4. Попробуем \( f(x) = x + c \): тогда \( f(f(x)+y) = f(x+c+y) = x+c+y+c = x+y+2c \), а \( x+f(y) = x+y+c \). Значит, \( 2c = c \) ⇒ \( c=0 \). \( f(x)=x \).

Шаг 5. Попробуем \( f(x) = -x + c \): \( f(f(x)+y) = f(-x+c+y) = -(-x+c+y)+c = x - y \). А \( x+f(y) = x - y + c \). Значит \( c=0 \). \( f(x)=-x \).

Ответ: \( f(x) = x \) или \( f(x) = -x \).

Задача 38. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(f(x) + y) = f(x) + f(y) \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(f(x)) = f(x) + f(0) \). (1)

Шаг 2. Подстановка \( x=0 \): \( f(f(0) + y) = f(0) + f(y) \). (2)

Шаг 3. Классическое решение: \( f(x) = 0 \) или \( f(x) = x + c \).

Ответ: \( f(x) \equiv 0 \) или \( f(x) = x + c \).

Задача 39. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(f(x) + y) = f(x) + f(y) + 1 \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( y=0 \): \( f(f(x)) = f(x) + f(0) + 1 \).

Шаг 2. Подстановка \( x=0 \): \( f(f(0)+y) = f(0)+f(y)+1 \).

Шаг 3. Попробуем \( f(x) = x + c \): \( f(f(x)+y) = f(x+c+y) = x+c+y+c = x+y+2c \). \( f(x)+f(y)+1 = x+c+y+c+1 = x+y+2c+1 \). Должно быть \( 2c = 2c+1 \) — противоречие.

Шаг 4. Попробуем \( f(x) = -x + c \): не совпадает. Значит, решений нет.

Ответ: решений нет.

Задача 40. Найдите все функции \( f: \mathbb{R} \to \mathbb{R} \), удовлетворяющие: \( f(x + f(y)) = f(x) + f(y) + x \) для всех \( x,y \in \mathbb{R} \).

Решение:

Шаг 1. Подстановка \( x=0 \): \( f(f(y)) = f(0) + f(y) \). (1)

Шаг 2. Подстановка \( y=0 \): \( f(x + f(0)) = f(x) + f(0) + x \). (2)

Шаг 3. Из (1): \( f(f(y)) - f(y) = f(0) \). Значит, \( f(t) - t = f(0) \) для всех \( t \) в образе \( f \). То есть \( f(t) = t + f(0) \) на образе.

Шаг 4. Пусть \( c = f(0) \). Тогда \( f(f(y)) = f(y) + c \).

Шаг 5. Подставим в исходное \( x = f(x) \): \( f(f(x) + f(y)) = f(f(x)) + f(y) + f(x) = f(x) + c + f(y) + f(x) = 2f(x) + f(y) + c \).

Шаг 6. С другой стороны, из симметрии: \( f(f(y) + f(x)) = 2f(y) + f(x) + c \). Сравнивая: \( 2f(x) + f(y) = 2f(y) + f(x) \) ⇒ \( f(x) = f(y) \) для всех \( x,y \). Значит, \( f \) постоянна. Пусть \( f(x) = k \). Тогда \( k = k + k + x \) ⇒ \( k = 2k + x \) для всех \( x \) — противоречие.

Ответ: решений нет.
