📖 Часть 1. Основные определения и свойства корней

1.1. Корень \(n\)-й степени

Определение: Корнем \(n\)-й степени из числа \(a\) называется такое число \(b\), что \(b^n = a\).

Для \(n\) нечётного: корень определён для всех \(a \in \mathbb{R}\).

Для \(n\) чётного: корень определён только для \(a \ge 0\), и значение корня неотрицательно (арифметический корень).

Примеры:

  • \(\sqrt[3]{-8} = -2\), потому что \((-2)^3 = -8\).
  • \(\sqrt{9} = 3\), а не \(\pm 3\) (арифметический корень).
  • \(\sqrt[4]{16} = 2\), потому что \(2^4 = 16\).

1.2. Основные свойства корней

1. Корень из произведения:

\[ \sqrt[n]{ab} = \sqrt[n]{a} \cdot \sqrt[n]{b}, \quad \text{если } a,b \ge 0 \text{ (для чётных } n\text{)}. \]

2. Корень из частного:

\[ \sqrt[n]{\frac{a}{b}} = \frac{\sqrt[n]{a}}{\sqrt[n]{b}}, \quad b \neq 0, \ a \ge 0 \text{ (для чётных } n\text{)}. \]

3. Возведение корня в степень:

\[ (\sqrt[n]{a})^k = \sqrt[n]{a^k}. \]

4. Корень из корня:

\[ \sqrt[m]{\sqrt[n]{a}} = \sqrt[mn]{a}. \]

5. Сокращение показателя:

\[ \sqrt[nk]{a^{mk}} = \sqrt[n]{a^m}, \quad \text{если } a \ge 0. \]

Примеры:

\(\sqrt[6]{64} = \sqrt[6]{2^6} = 2\).

\(\sqrt[4]{x^2} = \sqrt{|x|}\) для чётного корня!

⚠️ Важно: для чётных корней \(\sqrt[n]{x^n} = |x|\), а не \(x\). Для нечётных: \(\sqrt[n]{x^n} = x\).

🔧 Часть 2. Преобразование иррациональных выражений

2.1. Вынесение множителя из-под знака корня

Алгоритм:

1. Разложить подкоренное выражение на множители.

2. Представить каждый множитель как \(a^n \cdot b\), где \(a^n\) — полная степень.

3. Вынести \(a\) из-под корня.

Пример 1: \(\sqrt{48} = \sqrt{16 \cdot 3} = 4\sqrt{3}\).

Пример 2: \(\sqrt[3]{54} = \sqrt[3]{27 \cdot 2} = 3\sqrt[3]{2}\).

Пример 3 (с буквами): \(\sqrt{8a^5} = 2a^2\sqrt{2a}\) при \(a \ge 0\).

2.2. Внесение множителя под знак корня

Чтобы внести множитель под корень, его нужно возвести в степень корня:

\[ a\sqrt[n]{b} = \sqrt[n]{a^n b}. \]

Пример: \(3\sqrt{5} = \sqrt{45}\).

Пример: \(2\sqrt[3]{4} = \sqrt[3]{32}\).

⚠️ Внимание: если множитель отрицательный, а корень чётной степени, то вносить его нельзя без учёта знака (потому что \(\sqrt{a^2 b} = |a|\sqrt{b}\)).

2.3. Избавление от иррациональности в знаменателе

Случай 1: Знаменатель — одночлен \(\sqrt{a}\)

Умножаем числитель и знаменатель на \(\sqrt{a}\):

\[ \frac{1}{\sqrt{a}} = \frac{\sqrt{a}}{a}. \]

Пример: \(\frac{3}{\sqrt{2}} = \frac{3\sqrt{2}}{2}\).

Случай 2: Знаменатель — сумма или разность \(\sqrt{a} \pm \sqrt{b}\)

Используем сопряжённое выражение:

  • Для \(\sqrt{a} + \sqrt{b}\) сопряжённое — \(\sqrt{a} - \sqrt{b}\).
  • Для \(\sqrt{a} - \sqrt{b}\) сопряжённое — \(\sqrt{a} + \sqrt{b}\).

Формула: \((\sqrt{a} + \sqrt{b})(\sqrt{a} - \sqrt{b}) = a - b\).

Пример:

\[ \frac{1}{\sqrt{3} + \sqrt{2}} = \frac{\sqrt{3} - \sqrt{2}}{3 - 2} = \sqrt{3} - \sqrt{2}. \]

Случай 3: Знаменатель — \(\sqrt{a} \pm b\)

Сопряжённое: \(\sqrt{a} \mp b\). Формула: \((\sqrt{a} + b)(\sqrt{a} - b) = a - b^2\).

Пример: \(\frac{5}{\sqrt{2} - 1} = 5(\sqrt{2}+1)\).

Случай 4: Знаменатель — сумма трёх корней

Иногда нужно применять сопряжение дважды.

Пример: \(\frac{1}{\sqrt{2} + \sqrt{3} + \sqrt{5}}\).

Сначала группируем: \((\sqrt{2}+\sqrt{3}) + \sqrt{5}\).

Умножаем на сопряжённое \((\sqrt{2}+\sqrt{3}) - \sqrt{5}\):

\[ \frac{(\sqrt{2}+\sqrt{3}) - \sqrt{5}}{(\sqrt{2}+\sqrt{3})^2 - 5} = \frac{\sqrt{2}+\sqrt{3}-\sqrt{5}}{2\sqrt{6}}. \]

Теперь избавляемся от \(\sqrt{6}\) в знаменателе: умножаем на \(\sqrt{6}\):

\[ \frac{\sqrt{6}(\sqrt{2}+\sqrt{3}-\sqrt{5})}{12}. \]

2.4. Преобразование двойных радикалов

Двойной радикал — это выражение вида \(\sqrt{a \pm \sqrt{b}}\).

Формула:

\[ \sqrt{a + \sqrt{b}} = \sqrt{\frac{a + \sqrt{a^2 - b}}{2}} + \sqrt{\frac{a - \sqrt{a^2 - b}}{2}}. \] \[ \sqrt{a - \sqrt{b}} = \sqrt{\frac{a + \sqrt{a^2 - b}}{2}} - \sqrt{\frac{a - \sqrt{a^2 - b}}{2}}. \]

Условие применимости: \(a^2 - b\) должно быть полным квадратом.

Пример: \(\sqrt{3 + 2\sqrt{2}}\).

\(a = 3, \ b = (2\sqrt{2})^2 = 8, \ a^2 - b = 9 - 8 = 1\).

\[ \sqrt{\frac{3+1}{2}} + \sqrt{\frac{3-1}{2}} = \sqrt{2} + \sqrt{1} = \sqrt{2} + 1. \]

Проверка: \((\sqrt{2}+1)^2 = 3 + 2\sqrt{2}\). Верно!

✍️ Практика

Задача 1. Упростите \(\sqrt{72} + \sqrt{50} - \sqrt{98}\).

Решение:

\(\sqrt{72} = \sqrt{36\cdot2} = 6\sqrt{2}\).

\(\sqrt{50} = \sqrt{25\cdot2} = 5\sqrt{2}\).

\(\sqrt{98} = \sqrt{49\cdot2} = 7\sqrt{2}\).

Сумма: \(6\sqrt{2} + 5\sqrt{2} - 7\sqrt{2} = 4\sqrt{2}\).

Ответ: \(4\sqrt{2}\).

Задача 2. Избавьтесь от иррациональности в знаменателе: \(\frac{2}{\sqrt{5} - \sqrt{3}}\).

Решение:

Умножаем числитель и знаменатель на сопряжённое \(\sqrt{5} + \sqrt{3}\):

\[ \frac{2}{\sqrt{5} - \sqrt{3}} \cdot \frac{\sqrt{5} + \sqrt{3}}{\sqrt{5} + \sqrt{3}} = \frac{2(\sqrt{5} + \sqrt{3})}{5 - 3} = \sqrt{5} + \sqrt{3}. \]

Ответ: \(\sqrt{5} + \sqrt{3}\).

Задача 3. Упростите \(\sqrt{7 + 4\sqrt{3}}\).

Решение:

\(a = 7, \ b = (4\sqrt{3})^2 = 48, \ a^2 - b = 49 - 48 = 1\).

\[ \sqrt{7 + 4\sqrt{3}} = \sqrt{\frac{7+1}{2}} + \sqrt{\frac{7-1}{2}} = \sqrt{4} + \sqrt{3} = 2 + \sqrt{3}. \]

Ответ: \(2 + \sqrt{3}\).
