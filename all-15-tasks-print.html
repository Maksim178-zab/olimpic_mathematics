⚙️ Часть 4. Неравенства с параметрами — задачи 36–42

36. Система \(\begin{cases}x^2+2x+a\le0\\ x^2-4x-6a\le0\end{cases}\) имеет единственное решение. Найти \(a\).
37. \(x+\sqrt{x(a-x)}=1\) имеет хотя бы одно решение.
38. \(x+\sqrt{a-x^2}=1\) имеет два различных решения.
39. \(\frac{x^2+px+1}{x^2-x+1}\le3\) для всех \(x\). Найти \(p\).
40. \(|x-2a|+|x+2a|\le4a\) имеет хотя бы одно решение.
41. \((a-2)x^2+2(a-3)x+a-4<0\) для всех \(x>0\).
42. \(\begin{cases}x^2+y^2\le a\\ x+y\ge2\end{cases}\) имеет решение.
