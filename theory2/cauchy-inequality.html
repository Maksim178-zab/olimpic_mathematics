📐 Неравенство Коши

Среднее арифметическое ≥ среднего геометрического

🎯 Формулировка

Для любых неотрицательных чисел \(a_1, a_2, \dots, a_n\): \[\frac{a_1 + a_2 + \dots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \dots a_n}\] Равенство достигается, когда \(a_1 = a_2 = \dots = a_n\).
Для \(n=2\): \(\frac{a+b}{2} \ge \sqrt{ab}\), \(a,b \ge 0\).

📖 Доказательство для \(n=2\)

Для любых неотрицательных чисел \(a\) и \(b\) рассмотрим разность:

\[ (\sqrt{a} - \sqrt{b})^2 \ge 0. \]

Раскрываем скобки:

\[ a - 2\sqrt{ab} + b \ge 0 \quad\Rightarrow\quad a + b \ge 2\sqrt{ab}. \]

Делим обе части на 2:

\[ \frac{a+b}{2} \ge \sqrt{ab}. \]

Равенство достигается, когда \(\sqrt{a} = \sqrt{b}\), то есть \(a = b\).

📖 Доказательство для \(n=3\)

Для неотрицательных \(a,b,c\) воспользуемся тождеством:

\[ a^3 + b^3 + c^3 - 3abc = \frac12(a+b+c)\bigl((a-b)^2 + (b-c)^2 + (c-a)^2\bigr). \]

Правая часть неотрицательна, так как \(a+b+c \ge 0\) и сумма квадратов \(\ge 0\). Значит:

\[ a^3 + b^3 + c^3 \ge 3abc. \]

Теперь положим \(a = \sqrt[3]{x}\), \(b = \sqrt[3]{y}\), \(c = \sqrt[3]{z}\), где \(x,y,z \ge 0\). Тогда:

\[ x + y + z \ge 3\sqrt[3]{xyz}. \]

Делим на 3:

\[ \frac{x+y+z}{3} \ge \sqrt[3]{xyz}. \]

Равенство достигается при \(x = y = z\).

📖 Доказательство общего случая (через индукцию Коши)

Шаг 1. База: \(n=2\)

Неравенство для двух чисел уже доказано выше.

Шаг 2. Индукционный переход: от \(n\) к \(2n\)

Пусть неравенство Коши верно для любого набора из \(n\) неотрицательных чисел.

Докажем его для \(2n\) чисел. Пусть даны \(2n\) неотрицательных чисел:

\[ a_1, a_2, \dots, a_n, a_{n+1}, \dots, a_{2n}. \]

Разобьём их на две группы по \(n\) чисел и обозначим их средние арифметические:

\[ A = \frac{a_1 + a_2 + \dots + a_n}{n}, \qquad B = \frac{a_{n+1} + a_{n+2} + \dots + a_{2n}}{n}. \]

По предположению индукции, для каждой группы выполняется неравенство Коши:

\[ A \ge \sqrt[n]{a_1 a_2 \dots a_n}, \qquad B \ge \sqrt[n]{a_{n+1} a_{n+2} \dots a_{2n}}. \]

Теперь применим неравенство Коши для двух чисел к \(A\) и \(B\):

\[ \frac{A + B}{2} \ge \sqrt{AB}. \]

Но \(\frac{A+B}{2}\) — это в точности среднее арифметическое всех \(2n\) чисел:

\[ \frac{A+B}{2} = \frac{\frac{a_1+\dots+a_n}{n} + \frac{a_{n+1}+\dots+a_{2n}}{n}}{2} = \frac{a_1+\dots+a_{2n}}{2n}. \]

Теперь оценим произведение \(AB\) снизу:

\[ AB \ge \sqrt[n]{a_1 a_2 \dots a_n} \cdot \sqrt[n]{a_{n+1} a_{n+2} \dots a_{2n}} = \sqrt[n]{a_1 a_2 \dots a_{2n}}. \]

Следовательно:

\[ \sqrt{AB} \ge \sqrt[2n]{a_1 a_2 \dots a_{2n}}. \]

Объединяя всё вместе:

\[ \frac{a_1+\dots+a_{2n}}{2n} \ge \sqrt[2n]{a_1 a_2 \dots a_{2n}}. \]

Индукционный переход от \(n\) к \(2n\) доказан.

Шаг 3. Спуск от \(2^k\) к произвольному \(n\)

Мы доказали неравенство для всех \(n\), являющихся степенями двойки: \(n = 2, 4, 8, 16, \dots\).

Теперь нужно доказать для любого натурального \(n\).

Идея: добавим несколько фиктивных чисел, равных среднему арифметическому, чтобы общее количество стало степенью двойки.

Пусть даны \(n\) чисел: \(a_1, a_2, \dots, a_n\). Обозначим их среднее арифметическое:

\[ A = \frac{a_1 + a_2 + \dots + a_n}{n}. \]

Выберем такое \(m\), что \(n + m = 2^k\) для некоторого натурального \(k\) (это всегда возможно, так как степени двойки не ограничены сверху).

Добавим \(m\) фиктивных чисел, каждое из которых равно \(A\):

\[ a_{n+1} = a_{n+2} = \dots = a_{n+m} = A. \]

Теперь для этих \(2^k = n+m\) чисел неравенство Коши уже доказано:

\[ \frac{a_1 + a_2 + \dots + a_n + A + \dots + A}{2^k} \ge \sqrt[2^k]{a_1 a_2 \dots a_n \cdot A^m}. \]

Левая часть этого неравенства равна:

\[ \frac{a_1+\dots+a_n + m A}{2^k} = \frac{nA + mA}{2^k} = \frac{(n+m)A}{2^k} = A. \]

Значит:

\[ A \ge \sqrt[2^k]{a_1 a_2 \dots a_n \cdot A^m}. \]

Возводим обе части в степень \(2^k\):

\[ A^{2^k} \ge a_1 a_2 \dots a_n \cdot A^m. \]

Если \(A > 0\), делим обе части на \(A^m\):

\[ A^{2^k - m} \ge a_1 a_2 \dots a_n. \]

Но \(2^k - m = n\). Следовательно:

\[ A^n \ge a_1 a_2 \dots a_n. \]

Извлекаем корень \(n\)-й степени (все числа неотрицательны, можно):

\[ A \ge \sqrt[n]{a_1 a_2 \dots a_n}. \]

А это и есть неравенство Коши для \(n\) чисел.

Если \(A = 0\), то все \(a_i = 0\) (так как они неотрицательны), и неравенство тривиально.

✅ Доказательство завершено.

📖 Альтернативное доказательство (через выпуклость \(\ln x\))

Это доказательство короче, но требует знакомства с логарифмами и выпуклостью.

Шаг 1. Логарифмируем обе части

Нам нужно доказать:

\[ \frac{a_1 + a_2 + \dots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \dots a_n}. \]

Если все \(a_i > 0\), можно прологарифмировать обе части. Логарифм — монотонно возрастающая функция, поэтому знак неравенства не меняется:

\[ \ln\left( \frac{a_1 + a_2 + \dots + a_n}{n} \right) \ge \ln\left( \sqrt[n]{a_1 a_2 \dots a_n} \right). \]

Правая часть преобразуется:

\[ \ln\left( \sqrt[n]{a_1 a_2 \dots a_n} \right) = \frac{1}{n} \ln(a_1 a_2 \dots a_n) = \frac{\ln a_1 + \ln a_2 + \dots + \ln a_n}{n}. \]

Таким образом, достаточно доказать:

\[ \ln\left( \frac{a_1 + a_2 + \dots + a_n}{n} \right) \ge \frac{\ln a_1 + \ln a_2 + \dots + \ln a_n}{n}. \]

Шаг 2. Используем выпуклость логарифма

Рассмотрим функцию \(f(x) = \ln x\) на промежутке \((0, +\infty)\).

Её вторая производная:

\[ f''(x) = -\frac{1}{x^2} < 0 \quad \text{для всех } x > 0. \]

Значит, функция \(f(x) = \ln x\) является вогнутой (выпуклой вверх).

По неравенству Йенсена для вогнутой функции:

\[ f\left( \frac{x_1 + x_2 + \dots + x_n}{n} \right) \ge \frac{f(x_1) + f(x_2) + \dots + f(x_n)}{n}. \]

Подставляем \(f(x) = \ln x\) и \(x_i = a_i\):

\[ \ln\left( \frac{a_1 + a_2 + \dots + a_n}{n} \right) \ge \frac{\ln a_1 + \ln a_2 + \dots + \ln a_n}{n}. \]

Шаг 3. Потенцируем

Возводим \(e\) в обе части (экспонента — монотонно возрастающая функция):

\[ \frac{a_1 + a_2 + \dots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \dots a_n}. \]

✅ Альтернативное доказательство завершено.

Замечание: это доказательство использует неравенство Йенсена, поэтому оно не является «независимым», но оно очень наглядное и короткое. В олимпиадной практике его часто используют как «доказательство через выпуклость».

📐 Неравенство Коши в форме Шварца

\((\sum a_i b_i)^2 \le (\sum a_i^2)(\sum b_i^2)\). Равенство при пропорциональности.

Доказательство: Рассмотрим квадратный трёхчлен:

\[ f(t) = \sum (a_i t - b_i)^2 = t^2\sum a_i^2 - 2t\sum a_i b_i + \sum b_i^2 \ge 0 \quad \text{для всех } t. \]

Так как \(f(t) \ge 0\) для всех \(t\), дискриминант \(\le 0\):

\[ D = 4(\sum a_i b_i)^2 - 4(\sum a_i^2)(\sum b_i^2) \le 0. \]

Отсюда получаем требуемое неравенство.

Пример. Доказать \(\frac{x^2}{a}+\frac{y^2}{b} \ge \frac{(x+y)^2}{a+b}\) при \(a,b>0\).

По Шварцу: \((\frac{x}{\sqrt{a}}\sqrt{a} + \frac{y}{\sqrt{b}}\sqrt{b})^2 \le (\frac{x^2}{a}+\frac{y^2}{b})(a+b)\) ⇒ \((x+y)^2 \le (\frac{x^2}{a}+\frac{y^2}{b})(a+b)\).

📏 Метод касательной (неопределённых коэффициентов)

Чтобы доказать \(\sum f(a_i) \ge 0\) при \(\sum a_i = const\), строим касательную \(L(t)=f(t_0)+f'(t_0)(t-t_0)\) и доказываем \(f(t) \ge L(t)\).

Пример: \(a+b+c=3\), доказать \(a^2+b^2+c^2 \ge 3\).

\(f(t)=t^2\), \(f'(t)=2t\). Касательная в \(t_0=1\): \(L(t)=1+2(t-1)=2t-1\).

\(t^2-(2t-1)=(t-1)^2\ge0\). Суммируем: \(\sum a^2 \ge 2\sum a -3 = 6-3=3\).

✍️ Практика

Задача 1. \(\frac{x}{y}+\frac{y}{x} \ge 2\)

По Коши: \(\frac{x}{y}+\frac{y}{x} \ge 2\sqrt{1}=2\). Равенство при \(x=y\).

Задача 2. \(\frac{a}{b}+\frac{b}{c}+\frac{c}{a} \ge 3\)

По Коши для трёх чисел: \(\frac{\sum a/b}{3} \ge \sqrt[3]{1}=1\).

Задача 3 (олимп). \(\frac{a^2}{b}+\frac{b^2}{c}+\frac{c^2}{a} \ge a+b+c\)

\(\frac{a^2}{b}+b \ge 2a\), \(\frac{b^2}{c}+c \ge 2b\), \(\frac{c^2}{a}+a \ge 2c\). Складываем: сумма ≥ 2(a+b+c) ⇒ исходное ≥ a+b+c.
