📐 Неравенство Коши

От квадрата разности до олимпиадных задач

🔑 Самый главный факт

Для любого действительного числа \(t\): \(t^2 \ge 0\). Равенство только при \(t=0\).
Почему это важно? Потому что почти все неравенства превращаются в сумму квадратов. Пример: \(a^2 + b^2 \ge 2ab\) ⇔ \((a-b)^2 \ge 0\).

📐 Неравенство Коши (для двух чисел)

Для любых неотрицательных \(x, y\): \(\frac{x+y}{2} \ge \sqrt{xy}\). Равенство при \(x=y\).
Числовой пример: \(x=4, y=9\): \((4+9)/2 = 6.5\), \(\sqrt{36}=6\), \(6.5 \ge 6\).
Доказательство: \((\sqrt{x}-\sqrt{y})^2 \ge 0 \Rightarrow x+y-2\sqrt{xy} \ge 0 \Rightarrow x+y \ge 2\sqrt{xy}\).

⚡ Как применять Коши (пошагово)

Задача. Найти минимум \(x + \frac{4}{x}\) при \(x>0\).

Шаг 1. Оба слагаемых положительны.
Шаг 2. По Коши: \(\frac{x + \frac{4}{x}}{2} \ge \sqrt{x \cdot \frac{4}{x}} = \sqrt{4}=2\).
Шаг 3. Умножаем на 2: \(x + \frac{4}{x} \ge 4\).
Шаг 4. Равенство при \(x = \frac{4}{x}\) ⇒ \(x=2\).
Ответ: 4.

📏 Коши для трёх и более чисел

\(\frac{a_1+\dots+a_n}{n} \ge \sqrt[n]{a_1 \cdots a_n}\). Равенство при \(a_1=\dots=a_n\).

Задача. Доказать \((a+b)(b+c)(c+a) \ge 8abc\) для \(a,b,c>0\).

\(a+b \ge 2\sqrt{ab}\), \(b+c \ge 2\sqrt{bc}\), \(c+a \ge 2\sqrt{ca}\). Перемножаем: \((a+b)(b+c)(c+a) \ge 8\sqrt{a^2b^2c^2}=8abc\).

📐 Неравенство между четырьмя средними

\(\min \le \frac{n}{\sum 1/a_i} \le \sqrt[n]{\prod a_i} \le \frac{\sum a_i}{n} \le \sqrt{\frac{\sum a_i^2}{n}} \le \max\)
При \(a+b+c=3\): \(abc \le 1\), \(a^2+b^2+c^2 \ge 3\).

⚡ Неравенство Коши в форме Шварца

\((\sum a_i b_i)^2 \le (\sum a_i^2)(\sum b_i^2)\). Равенство при пропорциональности.

Пример. Доказать \(\frac{x^2}{a} + \frac{y^2}{b} \ge \frac{(x+y)^2}{a+b}\) при \(a,b>0\).

По Шварцу: \((\frac{x}{\sqrt{a}}\sqrt{a} + \frac{y}{\sqrt{b}}\sqrt{b})^2 \le (\frac{x^2}{a}+\frac{y^2}{b})(a+b)\) ⇒ \((x+y)^2 \le (\frac{x^2}{a}+\frac{y^2}{b})(a+b)\).

📏 Метод касательной (неопределённых коэффициентов)

Чтобы доказать \(\sum f(a_i) \ge 0\) при \(\sum a_i = const\), строим касательную \(L(t)=f(t_0)+f'(t_0)(t-t_0)\) и доказываем \(f(t) \ge L(t)\).

Пример. \(a+b+c=3\), доказать \(a^2+b^2+c^2 \ge 3\).

\(f(t)=t^2\), \(f'(t)=2t\). Касательная в \(t_0=1\): \(L(t)=1+2(t-1)=2t-1\). \(t^2-(2t-1)=(t-1)^2\ge0\). Суммируем: \(\sum a^2 \ge 2\sum a -3 = 6-3=3\).

✍️ Практика (с полными решениями)

1. \(\frac{x}{y}+\frac{y}{x} \ge 2\)

По Коши: \(\frac{x}{y}+\frac{y}{x} \ge 2\sqrt{1}=2\). Равенство при \(x=y\).

2. \(\frac{a}{b}+\frac{b}{c}+\frac{c}{a} \ge 3\)

По Коши для трёх чисел: \(\frac{\sum a/b}{3} \ge \sqrt[3]{1}=1\).

3 (олимп). \(\frac{a^2}{b}+\frac{b^2}{c}+\frac{c^2}{a} \ge a+b+c\)

\(\frac{a^2}{b}+b \ge 2a\), \(\frac{b^2}{c}+c \ge 2b\), \(\frac{c^2}{a}+a \ge 2c\). Складываем: сумма ≥ 2(a+b+c) ⇒ исходное ≥ a+b+c.

Коши, Шварц, касательная — основа олимпиадной алгебры
