📐 Бернулли, Йенсен, Шур

От индукции до выпуклости и симметризации

📖 Неравенство Бернулли

Для любого \(x \ge -1\) и натурального \(n\): \((1+x)^n \ge 1 + nx\). При \(0 < n < 1\) знак меняется на \(\le\).

Доказательство (по индукции):

База: \(n=1\): \((1+x)^1 = 1+x\) — верно.

Шаг: предположим, что \((1+x)^k \ge 1+kx\). Тогда

\[ (1+x)^{k+1} = (1+x)^k(1+x) \ge (1+kx)(1+x) = 1+(k+1)x + kx^2 \ge 1+(k+1)x. \]

Так как \(kx^2 \ge 0\) (при \(x \ge -1\) и \(k \ge 0\)), неравенство доказано.

Равенство при \(x=0\) или \(n=1\).

Числовой пример: \((1-0.1)^5 = 0.59049 \ge 1 - 0.5 = 0.5\).

Применение. Доказать \(2^n > n\).

\(2^n = (1+1)^n \ge 1 + n\cdot1 = n+1 > n\).

📐 Что такое выпуклая функция? (Очень просто)

Функция выпуклая, если её график изогнут вверх (как чашка \(x^2\)). Вогнутая — изогнута вниз (как крыша \(\sqrt{x}\)).
Признак: \(f''(x) \ge 0\) ⇒ выпуклая; \(f''(x) \le 0\) ⇒ вогнутая.
Примеры выпуклых: \(x^2, e^x, 1/x\) (при \(x>0\)).
Примеры вогнутых: \(\sqrt{x}, \ln x, \sin x\) на \((0,\pi)\).

📏 Неравенство Йенсена

Для выпуклой \(f\): \(f(\frac{x_1+\dots+x_n}{n}) \le \frac{f(x_1)+\dots+f(x_n)}{n}\).
Для вогнутой — знак \(\ge\).

Доказательство для \(n=2\) (определение выпуклости):

По определению выпуклой функции: для любых \(x_1, x_2\) и \(t \in [0,1]\)

\[ f(tx_1+(1-t)x_2) \le t f(x_1) + (1-t) f(x_2). \]

При \(t=\frac12\) получаем:

\[ f\left(\frac{x_1+x_2}{2}\right) \le \frac{f(x_1)+f(x_2)}{2}. \]

Для \(n\) чисел — обобщение по индукции с использованием выпуклости.

Для вогнутой функции все знаки меняются на противоположные.

Пример с квадратом (выпуклая): \(\left(\frac{a+b}{2}\right)^2 \le \frac{a^2+b^2}{2}\) ⇔ \((a-b)^2 \ge 0\).
Пример с корнем (вогнутая): \(\sqrt{\frac{a+b}{2}} \ge \frac{\sqrt{a}+\sqrt{b}}{2}\).

Олимпиадная задача. Для углов треугольника \(A+B+C=\pi\): \(\sin A+\sin B+\sin C \le \frac{3\sqrt{3}}{2}\).

\(\sin x\) вогнута на \((0,\pi)\). По Йенсену: \(\frac{\sin A+\sin B+\sin C}{3} \le \sin(\frac{A+B+C}{3}) = \sin(\pi/3)=\frac{\sqrt{3}}{2}\). Умножаем на 3.

🔴 Неравенство Шура (r=1)

\(a^3+b^3+c^3+3abc \ge \sum_{\text{sym}} a^2 b\) (где \(\sum_{\text{sym}} a^2 b = a^2b+a^2c+b^2a+b^2c+c^2a+c^2b\)).

Доказательство: Известно тождество:

\[ a^3+b^3+c^3+3abc - \sum_{\text{sym}} a^2 b = \frac12(a+b+c)\big[(a-b)^2+(b-c)^2+(c-a)^2\big] \ge 0. \]

Все множители неотрицательны (\(a,b,c \ge 0\)), значит, левая часть \(\ge 0\).

Равенство при \(a=b=c\).

Задача. Доказать \(a^3+b^3+c^3+3abc \ge ab(a+b)+bc(b+c)+ca(c+a)\).

Правая часть = \(\sum_{\text{sym}} a^2 b\). Это и есть неравенство Шура.

🔄 Симметризация и нормализация

Если неравенство однородно (степени левой и правой частей одинаковы), можно положить \(a+b+c=1\) или \(abc=1\) без потери общности.

Несбитт (классика). \(\frac{a}{b+c}+\frac{b}{c+a}+\frac{c}{a+b} \ge \frac{3}{2}\).

Положим \(a+b+c=1\), тогда \(b+c=1-a\). Неравенство: \(\sum \frac{a}{1-a} \ge \frac32\). Функция \(f(t)=\frac{t}{1-t}\) выпукла на \((0,1)\), по Йенсену получаем требуемое.

📋 Пошаговый план решения любого олимпиадного неравенства

1. Проверь однородность → нормализуй.
2. Попробуй перенести всё в одну сторону и выделить квадраты.
3. Если есть суммы и произведения → Коши.
4. Если есть степени и корни → Йенсен (выпуклость).
5. Если есть дроби \(\frac{a^2}{b}\) → Шварц.
6. Всегда указывай, когда достигается равенство.

✍️ Итоговая практика

1. \(\sqrt{x} + \sqrt{y} + \sqrt{z} \le \sqrt{3(x+y+z)}\)

Корень — вогнутая функция, по Йенсену: \(\frac{\sum \sqrt{x_i}}{3} \le \sqrt{\frac{\sum x_i}{3}}\).

2. \(a^3+b^3+c^3 \ge a^2b+b^2c+c^2a\) (при \(a,b,c \ge 0\))

Частный случай неравенства Шура или перестановочного неравенства.
