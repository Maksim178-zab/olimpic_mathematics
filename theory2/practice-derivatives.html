📖 Часть 1. Базовые задачи на правила дифференцирования

Задача 1. Найдите производную функции \( f(x) = 3x^5 - 2x^3 + 7x - 4 \).

Подробное решение:

Шаг 1. Применяем правило: производная суммы равна сумме производных. \( f'(x) = (3x^5)' - (2x^3)' + (7x)' - (4)' \).

Шаг 2. Правило для степени: \( (x^n)' = n x^{n-1} \).

  • \( (3x^5)' = 3 \cdot 5x^{4} = 15x^4 \).
  • \( (2x^3)' = 2 \cdot 3x^{2} = 6x^2 \).
  • \( (7x)' = 7 \cdot 1 = 7 \).
  • \( (4)' = 0 \) (производная константы равна нулю).

Шаг 3. Собираем: \( f'(x) = 15x^4 - 6x^2 + 7 \).

Ответ: \( f'(x) = 15x^4 - 6x^2 + 7 \).

Задача 2. Найдите производную функции \( f(x) = \sin x + \cos x - \operatorname{tg} x \).

Подробное решение:

Шаг 1. Производная суммы: \( f'(x) = (\sin x)' + (\cos x)' - (\operatorname{tg} x)' \).

Шаг 2. Табличные производные:

  • \( (\sin x)' = \cos x \).
  • \( (\cos x)' = -\sin x \).
  • \( (\operatorname{tg} x)' = \frac{1}{\cos^2 x} \).

Шаг 3. Подставляем: \( f'(x) = \cos x - \sin x - \frac{1}{\cos^2 x} \).

Ответ: \( f'(x) = \cos x - \sin x - \frac{1}{\cos^2 x} \).

Задача 3. Найдите производную функции \( f(x) = x^2 \ln x \).

Подробное решение:

Шаг 1. Это произведение двух функций: \( u = x^2 \), \( v = \ln x \).

Шаг 2. Правило произведения: \( (uv)' = u'v + uv' \).

Шаг 3. Находим: \( u' = 2x \), \( v' = \frac{1}{x} \).

Шаг 4. Подставляем: \( f'(x) = 2x \cdot \ln x + x^2 \cdot \frac{1}{x} = 2x \ln x + x \).

Ответ: \( f'(x) = 2x \ln x + x \).

Задача 4. Найдите производную функции \( f(x) = \frac{e^x}{x} \).

Подробное решение:

Шаг 1. Это частное двух функций: \( u = e^x \), \( v = x \).

Шаг 2. Правило частного: \( \left( \frac{u}{v} \right)' = \frac{u'v - uv'}{v^2} \).

Шаг 3. Находим: \( u' = e^x \), \( v' = 1 \).

Шаг 4. Подставляем: \( f'(x) = \frac{e^x \cdot x - e^x \cdot 1}{x^2} = \frac{e^x (x - 1)}{x^2} \).

Ответ: \( f'(x) = \frac{e^x (x - 1)}{x^2} \).

Задача 5. Найдите производную функции \( f(x) = \sin^2 x \).

Подробное решение:

Шаг 1. Это сложная функция: внешняя — \( t^2 \), внутренняя — \( \sin x \).

Шаг 2. Правило цепочки: \( (f(g(x)))' = f'(g(x)) \cdot g'(x) \).

Шаг 3. Внешняя производная: \( (t^2)' = 2t \). Внутренняя: \( (\sin x)' = \cos x \).

Шаг 4. Подставляем: \( f'(x) = 2\sin x \cdot \cos x = \sin 2x \).

Ответ: \( f'(x) = \sin 2x \).

Задача 6. Найдите производную функции \( f(x) = \ln(\cos x) \).

Подробное решение:

Шаг 1. Сложная функция: внешняя \( \ln t \), внутренняя \( \cos x \).

Шаг 2. \( (\ln t)' = \frac{1}{t} \), \( (\cos x)' = -\sin x \).

Шаг 3. Подставляем: \( f'(x) = \frac{1}{\cos x} \cdot (-\sin x) = -\operatorname{tg} x \).

Ответ: \( f'(x) = -\operatorname{tg} x \).

📊 Часть 2. Исследование функций: монотонность, экстремумы, выпуклость

Задача 7. Найдите промежутки возрастания и убывания функции \( f(x) = x^3 - 3x^2 + 2 \).

Подробное решение:

Шаг 1. Находим производную: \( f'(x) = 3x^2 - 6x = 3x(x-2) \).

Шаг 2. Находим критические точки: \( f'(x) = 0 \) ⇒ \( 3x(x-2) = 0 \) ⇒ \( x = 0 \), \( x = 2 \).

Шаг 3. Определяем знаки производной на интервалах:

  • На \( (-\infty, 0) \): \( x = -1 \), \( f'(-1) = 9 > 0 \) ⇒ возрастает.
  • На \( (0, 2) \): \( x = 1 \), \( f'(1) = -3 < 0 \) ⇒ убывает.
  • На \( (2, +\infty) \): \( x = 3 \), \( f'(3) = 9 > 0 \) ⇒ возрастает.
Ответ: возрастает на \( (-\infty, 0) \cup (2, +\infty) \), убывает на \( (0, 2) \).

Задача 8. Найдите точки экстремума функции \( f(x) = x^3 - 3x^2 + 2 \).

Подробное решение:

Шаг 1. Производная: \( f'(x) = 3x(x-2) \).

Шаг 2. Критические точки: \( x = 0 \), \( x = 2 \).

Шаг 3. Знаки производной: при переходе через \( x = 0 \) знак меняется с \( + \) на \( - \) ⇒ максимум. При переходе через \( x = 2 \) знак меняется с \( - \) на \( + \) ⇒ минимум.

Шаг 4. Значения: \( f(0) = 2 \), \( f(2) = -2 \).

Ответ: \( x = 0 \) — максимум (2), \( x = 2 \) — минимум (-2).

Задача 9. Найдите наибольшее и наименьшее значение функции \( f(x) = x^3 - 3x^2 + 2 \) на отрезке \( [-1, 3] \).

Подробное решение:

Шаг 1. Критические точки внутри отрезка: \( x = 0 \), \( x = 2 \).

Шаг 2. Вычисляем значения: \( f(-1) = -2 \), \( f(0) = 2 \), \( f(2) = -2 \), \( f(3) = 2 \).

Шаг 3. Сравниваем: наибольшее значение 2 (в точках \( x=0 \) и \( x=3 \)), наименьшее -2 (в точках \( x=-1 \) и \( x=2 \)).

Ответ: максимум 2, минимум -2.

Задача 10. Найдите промежутки выпуклости и точки перегиба функции \( f(x) = x^4 - 6x^2 + 3 \).

Подробное решение:

Шаг 1. Первая производная: \( f'(x) = 4x^3 - 12x \).

Шаг 2. Вторая производная: \( f''(x) = 12x^2 - 12 = 12(x-1)(x+1) \).

Шаг 3. \( f''(x) = 0 \) ⇒ \( x = \pm 1 \).

Шаг 4. Знаки \( f''(x) \):

  • На \( (-\infty, -1) \): \( f'' > 0 \) ⇒ выпукла вниз.
  • На \( (-1, 1) \): \( f'' < 0 \) ⇒ выпукла вверх.
  • На \( (1, +\infty) \): \( f'' > 0 \) ⇒ выпукла вниз.

Шаг 5. В точках \( x = \pm 1 \) вторая производная меняет знак ⇒ точки перегиба.

Ответ: выпукла вниз на \( (-\infty,-1)\cup(1,+\infty) \), выпукла вверх на \( (-1,1) \), точки перегиба \( x=\pm1 \).

Задача 11. Найдите количество корней уравнения \( x^3 - 3x + a = 0 \) в зависимости от параметра \( a \).

Подробное решение:

Шаг 1. \( f(x) = x^3 - 3x + a \). \( f'(x) = 3x^2 - 3 = 3(x-1)(x+1) \).

Шаг 2. Критические точки: \( x=-1 \) (максимум), \( x=1 \) (минимум).

Шаг 3. \( f(-1) = a+2 \), \( f(1) = a-2 \).

  • \( a < -2 \): максимум < 0, минимум < 0 ⇒ 1 корень.
  • \( a = -2 \): максимум = 0 ⇒ 2 корня.
  • \( -2 < a < 2 \): максимум > 0, минимум < 0 ⇒ 3 корня.
  • \( a = 2 \): минимум = 0 ⇒ 2 корня.
  • \( a > 2 \): минимум > 0 ⇒ 1 корень.
Ответ: \( a<-2 \) или \( a>2 \): 1 корень; \( a=\pm2 \): 2 корня; \( -2< a<2 \): 3 корня.

Задача 12. Найдите все значения параметра \( a \), при которых функция \( f(x) = x^3 + ax^2 + 3x - 1 \) имеет ровно одну критическую точку.

Подробное решение:

Шаг 1. \( f'(x) = 3x^2 + 2ax + 3 \).

Шаг 2. Чтобы была ровно одна критическая точка, дискриминант должен быть равен нулю: \( D = 4a^2 - 36 = 4(a^2 - 9) \).

Шаг 3. \( D = 0 \) ⇒ \( a = \pm 3 \).

Ответ: \( a = -3 \) или \( a = 3 \).

Задача 13. Найдите точки экстремума функции \( f(x) = \frac{x}{x^2+1} \).

Подробное решение:

Шаг 1. \( f'(x) = \frac{1-x^2}{(x^2+1)^2} \).

Шаг 2. \( f'(x) = 0 \) ⇒ \( x = \pm 1 \).

Шаг 3. Знаки: на \( (-\infty,-1) \) \( f'<0 \), на \( (-1,1) \) \( f'>0 \), на \( (1,\infty) \) \( f'<0 \).

Шаг 4. В \( x=-1 \) — минимум, в \( x=1 \) — максимум.

Шаг 5. \( f(-1) = -\frac{1}{2} \), \( f(1) = \frac{1}{2} \).

Ответ: минимум \( -\frac{1}{2} \) при \( x=-1 \), максимум \( \frac{1}{2} \) при \( x=1 \).

Задача 14. Найдите промежутки монотонности функции \( f(x) = e^{-x^2} \).

Подробное решение:

Шаг 1. \( f'(x) = -2x e^{-x^2} \).

Шаг 2. \( e^{-x^2} > 0 \) всегда, знак определяется знаком \( -2x \).

  • При \( x<0 \): \( -2x > 0 \) ⇒ \( f'(x) > 0 \) ⇒ возрастает.
  • При \( x>0 \): \( -2x < 0 \) ⇒ \( f'(x) < 0 \) ⇒ убывает.
Ответ: возрастает на \( (-\infty,0) \), убывает на \( (0,+\infty) \).

🏔️ Часть 3. Применение производной в оптимизации

Задача 15. Из всех прямоугольников с площадью 36 см² найдите прямоугольник с наименьшим периметром.

Подробное решение:

Шаг 1. \( xy = 36 \). Периметр \( P = 2(x+y) \).

Шаг 2. \( y = \frac{36}{x} \), \( P(x) = 2\left(x + \frac{36}{x}\right) \).

Шаг 3. \( P'(x) = 2\left(1 - \frac{36}{x^2}\right) = 0 \) ⇒ \( x = 6 \).

Шаг 4. При \( x<6 \): \( P'<0 \) (убывает), при \( x>6 \): \( P'>0 \) (возрастает) ⇒ минимум.

Шаг 5. \( y = 6 \), \( P = 24 \).

Ответ: квадрат 6×6, периметр 24 см.

Задача 16. Найдите наименьшее расстояние от точки \( (0, 3) \) до параболы \( y = x^2 \).

Подробное решение:

Шаг 1. Точка на параболе: \( (x, x^2) \). Расстояние: \( d = \sqrt{x^2 + (x^2-3)^2} \).

Шаг 2. Минимум \( d \) там же, где минимум \( f(x) = x^4 - 5x^2 + 9 \).

Шаг 3. \( f'(x) = 4x^3 - 10x = 2x(2x^2 - 5) \). Критические точки: \( x=0 \), \( x=\pm\sqrt{\frac{5}{2}} \).

Шаг 4. \( f(0)=9 \), \( f(\pm\sqrt{5/2}) = \frac{11}{4} \). Минимум \( \frac{11}{4} \).

Шаг 5. Расстояние: \( d = \frac{\sqrt{11}}{2} \).

Ответ: \( \frac{\sqrt{11}}{2} \).

Задача 17. Найдите максимальную площадь прямоугольника, вписанного в полукруг радиуса \( R \).

Подробное решение:

Шаг 1. Прямоугольник имеет координаты \( (\pm x, y) \) на полуокружности \( x^2 + y^2 = R^2 \). Площадь: \( S = 2xy = 2x\sqrt{R^2 - x^2} \).

Шаг 2. \( f(x) = S^2 = 4x^2(R^2 - x^2) = 4R^2x^2 - 4x^4 \).

Шаг 3. \( f'(x) = 8R^2x - 16x^3 = 8x(R^2 - 2x^2) \). \( f'(x)=0 \) ⇒ \( x = \frac{R}{\sqrt{2}} \).

Шаг 4. \( y = \frac{R}{\sqrt{2}} \), \( S_{\max} = 2 \cdot \frac{R}{\sqrt{2}} \cdot \frac{R}{\sqrt{2}} = R^2 \).

Ответ: \( R^2 \).

Задача 18. Найдите максимальный объём цилиндра, вписанного в конус с радиусом \( R \) и высотой \( H \).

Подробное решение:

Шаг 1. Из подобия: \( r = R\left(1 - \frac{h}{H}\right) \). \( V = \pi r^2 h = \pi R^2 h \left(1 - \frac{h}{H}\right)^2 \).

Шаг 2. Замена \( t = \frac{h}{H} \): \( V = \pi R^2 H \cdot t(1-t)^2 \).

Шаг 3. \( g(t) = t(1-t)^2 \), \( g'(t) = (3t-1)(t-1) \). Максимум при \( t = \frac{1}{3} \).

Шаг 4. \( V_{\max} = \frac{4\pi R^2 H}{27} \).

Ответ: \( \frac{4\pi R^2 H}{27} \).

Задача 19. Найдите наименьшее значение функции \( f(x) = x^2 + \frac{4}{x} \) при \( x > 0 \).

Подробное решение:

Шаг 1. \( f'(x) = 2x - \frac{4}{x^2} = \frac{2x^3 - 4}{x^2} \).

Шаг 2. \( f'(x)=0 \) ⇒ \( x^3 = 2 \) ⇒ \( x = \sqrt[3]{2} \).

Шаг 3. При \( x<\sqrt[3]{2} \): \( f'<0 \) (убывает), при \( x>\sqrt[3]{2} \): \( f'>0 \) (возрастает) ⇒ минимум.

Шаг 4. \( f(\sqrt[3]{2}) = 2^{2/3} + 4 \cdot 2^{-1/3} = 3 \cdot 2^{2/3} \).

Ответ: \( 3 \cdot 2^{2/3} \).

Задача 20. Найдите точку на графике \( y = \sqrt{x} \), ближайшую к точке \( (4, 0) \).

Подробное решение:

Шаг 1. Точка на графике: \( (x, \sqrt{x}) \). \( d^2 = (x-4)^2 + x = x^2 - 7x + 16 \).

Шаг 2. \( f(x) = x^2 - 7x + 16 \). \( f'(x) = 2x - 7 = 0 \) ⇒ \( x = \frac{7}{2} \).

Шаг 3. \( f''(x) = 2 > 0 \) ⇒ минимум.

Шаг 4. Точка: \( \left( \frac{7}{2}, \sqrt{\frac{7}{2}} \right) \).

Ответ: \( \left( \frac{7}{2}, \sqrt{\frac{7}{2}} \right) \).

Задача 21. Найдите наибольшее значение функции \( f(x) = \ln x - x \) на \( (0, +\infty) \).

Подробное решение:

Шаг 1. \( f'(x) = \frac{1}{x} - 1 = \frac{1-x}{x} \).

Шаг 2. \( f'(x)=0 \) ⇒ \( x=1 \).

Шаг 3. При \( x<1 \): \( f'>0 \) (возрастает), при \( x>1 \): \( f'<0 \) (убывает) ⇒ максимум.

Шаг 4. \( f(1) = -1 \).

Ответ: -1.

Задача 22. Найдите минимальное значение функции \( f(x) = e^x - x \).

Подробное решение:

Шаг 1. \( f'(x) = e^x - 1 = 0 \) ⇒ \( x = 0 \).

Шаг 2. При \( x<0 \): \( f'<0 \) (убывает), при \( x>0 \): \( f'>0 \) (возрастает) ⇒ минимум.

Шаг 3. \( f(0) = 1 \).

Ответ: 1.

🏆 Часть 4. Олимпиадные задачи с параметрами и доказательства

Задача 23. Докажите, что \( e^x \ge 1 + x \) для всех \( x \in \mathbb{R} \).

Подробное решение:

Шаг 1. Рассмотрим \( f(x) = e^x - 1 - x \).

Шаг 2. \( f'(x) = e^x - 1 \). \( f'(x)=0 \) ⇒ \( x=0 \).

Шаг 3. При \( x<0 \): \( f'<0 \) (убывает), при \( x>0 \): \( f'>0 \) (возрастает) ⇒ минимум в \( x=0 \).

Шаг 4. \( f(0) = 0 \). Значит, \( f(x) \ge 0 \) для всех \( x \).

Вывод: \( e^x \ge 1 + x \).

Задача 24. Докажите, что \( \ln x \le x - 1 \) для всех \( x > 0 \).

Подробное решение:

Шаг 1. Рассмотрим \( f(x) = x - 1 - \ln x \).

Шаг 2. \( f'(x) = 1 - \frac{1}{x} = \frac{x-1}{x} \).

Шаг 3. \( f'(x)=0 \) ⇒ \( x=1 \). При \( x<1 \): \( f'<0 \) (убывает), при \( x>1 \): \( f'>0 \) (возрастает) ⇒ минимум в \( x=1 \).

Шаг 4. \( f(1) = 0 \). Значит, \( f(x) \ge 0 \) для всех \( x>0 \).

Вывод: \( \ln x \le x - 1 \).

Задача 25. Найдите все значения параметра \( a \), при которых уравнение \( x^3 - 3x = a \) имеет ровно три корня.

Подробное решение:

Шаг 1. \( f(x) = x^3 - 3x \). \( f'(x) = 3x^2 - 3 = 3(x-1)(x+1) \).

Шаг 2. Экстремумы: \( x=-1 \) — максимум \( f(-1)=2 \), \( x=1 \) — минимум \( f(1)=-2 \).

Шаг 3. Чтобы уравнение имело три корня, прямая \( y=a \) должна пересекать график в трёх точках. Это возможно при \( -2 < a < 2 \).

Шаг 4. При \( a=\pm2 \) — два корня, при других \( a \) — один корень.

Ответ: \( -2 < a < 2 \).

Задача 26. Найдите все значения \( a \), при которых уравнение \( e^x = ax^2 \) имеет ровно два корня.

Подробное решение:

Шаг 1. \( a = \frac{e^x}{x^2} \), \( x \neq 0 \). Рассмотрим \( f(x) = \frac{e^x}{x^2} \).

Шаг 2. \( f'(x) = \frac{e^x(x-2)}{x^3} \). Критическая точка: \( x=2 \). \( f(2) = \frac{e^2}{4} \).

Шаг 3. На \( (-\infty,0) \) функция убывает от \( +\infty \) до \( 0^+ \). На \( (0,2) \) убывает от \( +\infty \) до \( e^2/4 \). На \( (2,+\infty) \) возрастает от \( e^2/4 \) до \( +\infty \).

Шаг 4. Чтобы было ровно два корня, нужно \( a = \frac{e^2}{4} \) (касание в \( x=2 \) плюс один отрицательный корень).

Ответ: \( a = \frac{e^2}{4} \).

Задача 27. Найдите все значения \( a \), при которых функция \( f(x) = x^3 + ax^2 + x + 1 \) возрастает на всей числовой прямой.

Подробное решение:

Шаг 1. \( f'(x) = 3x^2 + 2ax + 1 \).

Шаг 2. Чтобы функция возрастала на всей прямой, нужно \( f'(x) \ge 0 \) для всех \( x \).

Шаг 3. Дискриминант: \( D = 4a^2 - 12 \le 0 \) ⇒ \( a^2 \le 3 \) ⇒ \( -\sqrt{3} \le a \le \sqrt{3} \).

Ответ: \( [-\sqrt{3}, \sqrt{3}] \).

Задача 28. Найдите наименьшее значение выражения \( x^2 + y^2 \) при условии \( xy = 1 \).

Подробное решение:

Шаг 1. \( y = \frac{1}{x} \), \( x \neq 0 \). \( f(x) = x^2 + \frac{1}{x^2} \).

Шаг 2. Замена \( t = x^2 > 0 \): \( f = t + \frac{1}{t} \).

Шаг 3. \( f'(t) = 1 - \frac{1}{t^2} = 0 \) ⇒ \( t = 1 \).

Шаг 4. При \( t<1 \): \( f'<0 \) (убывает), при \( t>1 \): \( f'>0 \) (возрастает) ⇒ минимум.

Шаг 5. \( f(1) = 2 \).

Ответ: 2.

Задача 29. Найдите максимальный объём конуса, вписанного в шар радиуса \( R \).

Подробное решение:

Шаг 1. Осевое сечение: окружность радиуса \( R \) с вписанным равнобедренным треугольником. Высота конуса \( h \), радиус основания \( r \). \( r^2 = 2Rh - h^2 \).

Шаг 2. Объём: \( V = \frac{1}{3}\pi r^2 h = \frac{1}{3}\pi (2Rh^2 - h^3) \).

Шаг 3. \( V'(h) = \frac{1}{3}\pi (4Rh - 3h^2) = \frac{1}{3}\pi h(4R - 3h) = 0 \) ⇒ \( h = \frac{4R}{3} \).

Шаг 4. При \( h < 4R/3 \): \( V' > 0 \), при \( h > 4R/3 \): \( V' < 0 \) ⇒ максимум.

Шаг 5. \( V_{\max} = \frac{32\pi R^3}{81} \).

Ответ: \( \frac{32\pi R^3}{81} \).

Задача 30. Докажите, что для всех \( x > 0 \) выполняется \( \sin x < x \).

Подробное решение:

Шаг 1. Рассмотрим \( f(x) = x - \sin x \). \( f'(x) = 1 - \cos x \).

Шаг 2. \( f'(x) \ge 0 \) при всех \( x \), так как \( \cos x \le 1 \). Значит, \( f \) возрастает на \( [0, +\infty) \).

Шаг 3. \( f(0) = 0 \). Так как функция возрастает, для \( x > 0 \) имеем \( f(x) > f(0) = 0 \).

Шаг 4. Значит, \( x - \sin x > 0 \) ⇒ \( \sin x < x \).

Вывод: \( \sin x < x \) для всех \( x > 0 \).
