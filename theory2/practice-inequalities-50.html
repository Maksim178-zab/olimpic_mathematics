50 задач с полными, пошаговыми решениями

📐 Часть 1. Метод интервалов (задачи 1–10)

Задача 1. \((x+1)(x-2)(x-4) > 0\)

Решение пошагово:

1. Находим нули каждого множителя: \(x=-1\), \(x=2\), \(x=4\).

2. Отмечаем точки на числовой прямой в порядке: \(-1, 2, 4\). Интервалы: \((-\infty,-1), (-1,2), (2,4), (4,+\infty)\).

3. Определяем знак на каждом интервале (подстановкой):
при \(x=-2\): \((-)\cdot(-)\cdot(-) = -\)
при \(x=0\): \((+)\cdot(-)\cdot(-) = +\)
при \(x=3\): \((+)\cdot(+)\cdot(-) = -\)
при \(x=5\): \((+)\cdot(+)\cdot(+) = +\)

4. Нужно \(>0\) ⇒ выбираем интервалы со знаком «+»: \((-1,2)\) и \((4,+\infty)\).

Ответ: \((-1,2) \cup (4,+\infty)\)

Задача 2. \(\frac{x-2}{x+3} \le 0\)

Решение:

Нули числителя: \(x=2\) (включаем, так как ≤). Нули знаменателя: \(x=-3\) (выкалываем).

Интервалы: \((-\infty,-3), (-3,2], [2,+\infty)\). Подстановка: \(x=-4\) ⇒ \((-)/(-)=+\); \(x=0\) ⇒ \((-)/(+)=-\); \(x=3\) ⇒ \((+)/(+)=+\).

Нужно ≤0 ⇒ берём интервал со знаком «−» и включаем точку \(x=2\).

Ответ: \((-3,2]\)

Задача 3. \(\frac{(x-2)^2(x+1)}{x-4} \ge 0\)

Решение: Нули числителя: \(x=2\) (чётная степень), \(x=-1\). Нуль знаменателя: \(x=4\) (выколот).

Правый интервал \((4,+\infty)\): \(x=5\) ⇒ \((+)\cdot(+)/(+)=+\). Переход через 4 (нечётная) ⇒ знак меняется: на \((2,4)\): \(-\). Переход через 2 (чётная) ⇒ знак НЕ меняется: на \((-1,2)\): \(-\). Переход через -1 (нечётная) ⇒ знак меняется: на \((-\infty,-1)\): \(+\) .

Нужно ≥0 ⇒ берём интервалы с «+» и включаем нули числителя: \((-\infty,-1] \cup \{2\} \cup (4,+\infty)\).

Ответ: \((-\infty,-1] \cup \{2\} \cup (4,+\infty)\)

Задача 4. \(\frac{x^2+4}{x-1} > 0\)

Решение: Числитель \(x^2+4 >0\) при всех \(x\). Значит, знак дроби совпадает со знаком знаменателя. \(x-1>0 \Rightarrow x>1\).

Ответ: \((1,+\infty)\)

Задача 5. \(\frac{(x-1)(x+2)}{(x-3)(x+5)} \le 0\)

Решение: Нули числителя: \(1,-2\) (включаем). Нули знаменателя: \(3,-5\) (выкалываем).

Точки в порядке: \(-5,-2,1,3\). Подстановка даёт знаки: \(+,-,+,-,+\) слева направо.

Нужно ≤0 ⇒ интервалы со знаком «−»: \((-5,-2]\) и \([1,3)\).

Ответ: \((-5,-2] \cup [1,3)\)

Задача 6. \(\frac{2}{x} + \frac{3}{x-2} < 0\)

Решение: Приводим к общему знаменателю: \(\frac{2(x-2)+3x}{x(x-2)} = \frac{5x-4}{x(x-2)} < 0\).

Нули числителя: \(x=0.8\). Нули знаменателя: \(0,2\) (выколоты). Интервалы: \((-\infty,0), (0,0.8), (0.8,2), (2,\infty)\). Подстановка даёт знаки: \(+,-,+,-\)? Уточним: при \(x=0.5\) числитель отрицательный, знаменатель \(0.5\cdot(-1.5)=-0.75<0\) ⇒ \((-)/(-)=+\). На \((0.8,2)\): \(x=1\) ⇒ числитель 1>0, знаменатель \(1\cdot(-1)=-1<0\) ⇒ \((+)/(-)=-\). Значит, знак «−» только на \((0.8,2)\).

Ответ: \((0.8,2)\)

Задача 7. \(\frac{x^3-9x}{x^2-4} \ge 0\)

Решение: \(x^3-9x = x(x-3)(x+3)\), знаменатель \((x-2)(x+2)\).

Нули числителя: \(-3,0,3\) (включаем). Нули знаменателя: \(-2,2\) (выкалываем).

Точки: \(-3,-2,0,2,3\). Чередование: самый правый интервал \((3,\infty)\) знак + ⇒ \((2,3)\): –, \((0,2)\): +, \((-2,0)\): –, \((-3,-2)\): +, \((-\infty,-3)\): –.

Нужно ≥0 ⇒ берём «+» и включаем нули числителя: \([-3,-2) \cup [0,2) \cup [3,\infty)\).

Ответ: \([-3,-2) \cup [0,2) \cup [3,\infty)\)

Задача 8. \(|x-1| + |x-4| < 5\)

Решение: Точки смены знака: \(x=1,4\).

Случай 1: \(x \le 1\): \(|x-1|=1-x,\ |x-4|=4-x\). Сумма \(5-2x <5 \Rightarrow x>0\) ⇒ \((0,1]\).

Случай 2: \(1 \le x \le 4\): сумма \(3<5\) ⇒ весь \([1,4]\).

Случай 3: \(x \ge 4\): сумма \(2x-5<5 \Rightarrow x<5\) ⇒ \([4,5)\).

Объединяем: \((0,5)\).

Ответ: \((0,5)\)

Задача 9. \(\frac{2^x - 4}{x-2} > 0\)

Решение: \(2^x-4=0 \Rightarrow x=2\), но \(x=2\) выколот (знаменатель). При \(x<2\): \(2^x-4<0\), знаменатель \(<0\) ⇒ дробь >0. При \(x>2\): оба >0 ⇒ дробь >0.

Дробь положительна при всех \(x \neq 2\).

Ответ: \((-\infty,2) \cup (2,\infty)\)

Задача 10. \(\frac{\ln(x-2)}{x-3} \ge 0\)

Решение: ОДЗ: \(x>2,\ x\neq3\). \(\ln(x-2)=0\) при \(x=3\) (не входит).

На \((2,3)\): \(\ln(x-2)<0\), \(x-3<0\) ⇒ дробь \(>0\). На \((3,\infty)\): оба >0 ⇒ дробь \(>0\).

Ответ: \((2,3) \cup (3,\infty)\)

📏 Часть 2. Классические неравенства (Коши, Бернулли, Йенсен) — задачи 11–25

Задача 11. Докажите \(a+\frac1a \ge 2\) для \(a>0\).

\((\sqrt{a} - 1/\sqrt{a})^2 \ge 0 \Rightarrow a-2+1/a \ge 0 \Rightarrow a+1/a \ge 2\). Равенство при \(a=1\).

Задача 12. \(\frac{a+b+c}{3} \ge \sqrt[3]{abc}\) для \(a,b,c\ge0\).

Из тождества \(a^3+b^3+c^3-3abc=\frac12(a+b+c)[(a-b)^2+(b-c)^2+(c-a)^2]\ge0\). Подстановка \(a\to\sqrt[3]{a},b\to\sqrt[3]{b},c\to\sqrt[3]{c}\) даёт требуемое.

Задача 13. \((a+b)(b+c)(c+a) \ge 8abc\).

\(a+b\ge2\sqrt{ab},\ b+c\ge2\sqrt{bc},\ c+a\ge2\sqrt{ca}\). Перемножаем: \((a+b)(b+c)(c+a)\ge8\sqrt{a^2b^2c^2}=8abc\).

Задача 14. \(\frac{a}{b}+\frac{b}{a}\ge2\).

По Коши: \(\frac{a/b+b/a}{2}\ge\sqrt{1}=1\).

Задача 15. Найти минимум \(x+16/x,\ x>0\).

По Коши: \(\frac{x+16/x}{2}\ge\sqrt{16}=4\Rightarrow x+16/x\ge8\). Минимум 8 при \(x=4\).

Задача 16. Докажите неравенство Бернулли: \((1+x)^n\ge1+nx\) при \(x\ge-1\).

По индукции: база \(n=1\) верно. Шаг: \((1+x)^{k+1}=(1+x)^k(1+x)\ge(1+kx)(1+x)=1+(k+1)x+kx^2\ge1+(k+1)x\).

Задача 17. Докажите \(2^n>n\) для всех натуральных \(n\).

\(2^n=(1+1)^n\ge1+n>n\).

Задача 18. \(\left(\frac{a+b}{2}\right)^2 \le \frac{a^2+b^2}{2}\).

\(\frac{a^2+b^2}{2}-\left(\frac{a+b}{2}\right)^2 = \frac{(a-b)^2}{4}\ge0\).

Задача 19. \(\sqrt{\frac{a+b}{2}} \ge \frac{\sqrt{a}+\sqrt{b}}{2}\).

Возводим в квадрат: \(\frac{a+b}{2}\ge\frac{a+b+2\sqrt{ab}}{4}\Rightarrow 2(a+b)\ge a+b+2\sqrt{ab}\Rightarrow a+b\ge2\sqrt{ab}\) — верно.

Задача 20. Для \(a^2+b^2+c^2=1\) докажите \(\frac{a}{b+c}+\frac{b}{c+a}+\frac{c}{a+b}\ge\frac32\).

Неравенство Несбитта не зависит от масштаба, достаточно доказать для любых положительных. Одно из доказательств: \(\sum\frac{a}{b+c}=\sum\frac{a^2}{a(b+c)}\ge\frac{(a+b+c)^2}{2(ab+bc+ca)}\ge\frac{3(ab+bc+ca)}{2(ab+bc+ca)}=\frac32\).

Задача 21. \(a+b+c=1\), доказать \(\frac{a}{1+a}+\frac{b}{1+b}+\frac{c}{1+c}\le\frac34\).

Функция \(f(t)=\frac{t}{1+t}\) вогнута, по Йенсену: \(\frac13\sum f(a_i)\le f(\frac13)=\frac14\). Умножаем на 3.

Задача 22. \(x^2+y^2\ge2xy\).

\((x-y)^2\ge0\).

Задача 23. \(\frac{a^2}{b}+\frac{b^2}{c}+\frac{c^2}{a}\ge a+b+c\).

По Коши–Шварца: \((\sum\frac{a^2}{b})(\sum b)\ge (a+b+c)^2\) ⇒ \(\sum\frac{a^2}{b}\ge a+b+c\).

Задача 24. \(x^2+y^2+z^2\ge xy+yz+zx\).

\(2\sum x^2-2\sum xy=\sum (x-y)^2\ge0\).

Задача 25. \(xyz=1\), доказать \(x+y+z\ge3\).

По Коши: \(\frac{x+y+z}{3}\ge\sqrt[3]{xyz}=1\).

🏆 Часть 3. Продвинутые олимпиадные неравенства (Шур, Мюрхед, Несбитт) — задачи 26–35

Задача 26 (Шур). \(a^3+b^3+c^3+3abc \ge \sum_{\text{sym}} a^2b\).

Известное тождество: \(\frac12(a+b+c)[(a-b)^2+(b-c)^2+(c-a)^2]\ge0\).

Задача 27 (Мюрхед). \(a^3+b^3+c^3 \ge a^2b+b^2c+c^2a\) (при \(a\ge b\ge c\)).

По перестановочному неравенству или из \(a^3+b^3\ge a^2b+ab^2\) и циклического суммирования.

Задача 28 (IMO 2001). \(\frac{a}{\sqrt{a^2+8bc}}+\frac{b}{\sqrt{b^2+8ca}}+\frac{c}{\sqrt{c^2+8ab}}\ge1\).

Замена \(x=\frac{a}{\sqrt{a^2+8bc}}\) и т.д., затем применение неравенства Гёльдера или \(x^2=\frac{a^2}{a^2+8bc}\) ⇒ \(\frac{bc}{a^2}=\frac{1-x^2}{8x^2}\). Перемножение даёт \((1-x^2)(1-y^2)(1-z^2)=512x^2y^2z^2\), откуда следует \(x+y+z\ge1\).

Задача 29 (Несбитт). \(\frac{a}{b+c}+\frac{b}{c+a}+\frac{c}{a+b}\ge\frac32\).

\(\sum\frac{a}{b+c}=\sum\frac{a^2}{a(b+c)}\ge\frac{(a+b+c)^2}{2(ab+bc+ca)}\ge\frac{3(ab+bc+ca)}{2(ab+bc+ca)}=\frac32\).

Задача 30. \(\frac{a^2}{b^2}+\frac{b^2}{c^2}+\frac{c^2}{a^2}\ge\frac{a}{c}+\frac{b}{a}+\frac{c}{b}\).

\(\frac{a^2}{b^2}+\frac{b^2}{c^2}\ge2\frac{a}{c}\) и т.д., сложить.

Задача 31 (Караматы). Сформулировать теорему и пример.

Теорема: если \(\varphi\) выпукла и \((x_i)\) мажорирует \((y_i)\), то \(\sum\varphi(x_i)\ge\sum\varphi(y_i)\). Пример: для фиксированной суммы минимум суммы квадратов при равных переменных.

Задача 32. \((a+b)(b+c)(c+a)\ge\frac89(a+b+c)(ab+bc+ca)\).

Левая часть = \((a+b+c)(ab+bc+ca)-abc\). Неравенство равносильно \(\frac19(a+b+c)(ab+bc+ca)\ge abc\), что верно по Коши.

Задача 33 (параметр). \(\frac{x+3a-4}{x+a}>0\) для всех \(x\in[1;3]\). Найти \(a\).

Анализ расположения нуля \(4-3a\) и точки разрыва \(-a\) относительно отрезка даёт \(a\in(-\infty,-3)\cup(1,\infty)\).

Задача 34. Прямоугольный треугольник: \(a^4+a^2b^2+b^4\ge\frac34c^4\).

\(c^2=a^2+b^2\), \(L=c^4-a^2b^2\ge\frac34c^4\iff \frac14c^4\ge a^2b^2\iff c^2\ge2ab\) — верно.

Задача 35 (IMO 2001, уже была). см. задачу 28.

Решение аналогично задаче 28.

⚙️ Часть 4. Неравенства с параметрами — задачи 36–42

Задача 36. Система \(x^2+2x+a\le0,\ x^2-4x-6a\le0\) имеет единственное решение. Найти \(a\).

Приравниваем правый конец одного отрезка к левому другого: \(-1+\sqrt{1-a}=2-\sqrt{4+6a}\). Решение даёт \(a=0\) или \(a=48/49\). Проверка: оба подходят.

Задача 37. \(x+\sqrt{x(a-x)}=1\) имеет хотя бы одно решение.

ОДЗ \(x\le1\). Возводим в квадрат: \(2x^2-(a+2)x+1=0\). Дискриминант \(D=a^2+4a-4\ge0\Rightarrow a\le-2-2\sqrt2\) или \(a\ge-2+2\sqrt2\). С учётом \(x\le1\) и \(x(a-x)\ge0\) получаем \(a\ge2\sqrt2-2\).

Задача 38. \(x+\sqrt{a-x^2}=1\) имеет два различных решения.

Аналогично: \(2x^2-2x+(1-a)=0\), \(D=8a-4>0\Rightarrow a>1/2\). Корни \(\frac{1\pm\sqrt{2a-1}}{2}\). Условия \(x\le1\) и \(|x|\le\sqrt{a}\) дают \(a\le1\).

Задача 39. \(\frac{x^2+px+1}{x^2-x+1}\le3\) для всех \(x\).

Знаменатель >0 ⇒ \(x^2+px+1\le3x^2-3x+3\Rightarrow 2x^2-(p+3)x+2\ge0\) для всех \(x\) ⇒ дискриминант \(\le0\) ⇒ \((p+3)^2-16\le0\Rightarrow -7\le p\le1\).

Задача 40. \(|x-2a|+|x+2a|\le4a\) имеет решение.

Левая часть ≥ \(4|a|\), нужно \(4|a|\le4a\Rightarrow a\ge0\). При \(a\ge0\) отрезок \([-2a,2a]\) даёт равенство.

Задача 41. \((a-2)x^2+2(a-3)x+a-4<0\) для всех \(x>0\).

При \(a=2\): \(-2x-2<0\) — подходит. При \(a>2\) ветви вверх ⇒ нет. При \(a<2\) ветви вниз, но дискриминант всегда 4>0, корни положительны ⇒ между ними функция >0. Только \(a=2\).

Задача 42. Система \(x^2+y^2\le a,\ x+y\ge2\) имеет решение.

Расстояние от центра до прямой \(x+y=2\) равно \(\sqrt2\). Нужно \(\sqrt{a}\ge\sqrt2\Rightarrow a\ge2\).

🏁 Часть 5. Экстремальные и текстовые задачи — задачи 43–50

Задача 43. Среди прямоугольников с периметром \(P\) найти наибольшую площадь.

\(x+y=P/2\), \(S=xy\le\left(\frac{x+y}{2}\right)^2=\frac{P^2}{16}\). Равенство при \(x=y=P/4\) (квадрат).

Задача 44 (текстовая). Четыре трубы. Время заполнения: первая+третья 40 мин, вторая+четвёртая 50 мин. Найти max и min времени двумя любыми трубами.

Пропускные способности: \(a=0.005,\ b=0.01,\ c=0.02,\ d=0.01\) (в долях объёма в минуту). Наименьшая сумма \(0.005+0.01=0.015\) ⇒ время \(200/3\) мин, наибольшая \(0.02+0.01=0.03\) ⇒ время \(100/3\) мин.

Задача 45. Найти минимум \(x+1/x\) на \((0,\infty)\).

По Коши: \(x+1/x\ge2\sqrt1=2\). Равенство при \(x=1\).

Задача 46. \(a+b+c=1\), \(a,b,c>0\). Найти максимум \(abc\).

По Коши: \(\frac{a+b+c}{3}\ge\sqrt[3]{abc}\Rightarrow abc\le\frac1{27}\). Равенство при \(a=b=c=1/3\).

Задача 47. Несбитт (уже была).

См. задачу 29.

Задача 48. \(\cos\left(\frac32-4x-x^2\right)\ge0\) на интервале \((-83/20,0)\).

\(t=\frac{11}{2}-(x+2)^2\). При \(x\in(-4.15,0)\) \(t\in(0.88,1.5]\). \(\cos t\ge0\) при \(t\in[-1.57,1.57]\) ⇒ \(t\le\pi/2\approx1.57\) автоматически. Нужно \(t\ge-\pi/2\) ⇒ \((x+2)^2\le\frac{11+\pi}{2}\). Решение: \(|x+2|\le\sqrt{\frac{11+\pi}{2}}\) с учётом интервала.

Задача 49. Точки \(k\sqrt2\) на прямой. Верно ли, что длина отрезка станет постоянной?

Нет, не верно. Из-за иррациональности \(\sqrt2\) расстояния принимают два значения, процесс не стабилизируется.

Задача 50. \(x^2+y^2+z^2=1\), найти максимум \(xy+yz+zx\).

\(1+2S=(x+y+z)^2\ge0\Rightarrow S\ge-1/2\). Из \(x^2+y^2\ge2xy\) ⇒ \(1\ge2(xy+yz+zx)=2S\Rightarrow S\le\frac12\). Максимум \(1/2\) при \(x=y=z=1/\sqrt3\).
