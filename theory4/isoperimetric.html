📖 Легенда о царице Дидоне

Самая известная изопериметрическая задача родилась из легенды.

Царица Дидона, основательница Карфагена, получила от местного вождя разрешение взять столько земли, сколько можно охватить бычьей шкурой. Она разрезала шкуру на тончайшие полоски, связала их в длинную верёвку и… очертила полукруг, прижав его диаметр к морскому берегу.

Почему полукруг? Морская граница была «бесплатной» — её не нужно было огораживать верёвкой. Поэтому полукруг позволил захватить максимальную площадь при заданной длине верёвки. Это и есть изопериметрическая задача с дополнительным условием (часть границы фиксирована).

«В мире не происходит ничего, в чём бы не был виден смысл какого-нибудь максимума или минимума» — Леонард Эйлер

🎯 Изопериметрическая задача

Классическая формулировка: среди всех замкнутых кривых заданной длины \(L\) найти ту, которая ограничивает максимальную площадь \(A\).
Ответ: окружность.

Почему это не очевидно?

Этот результат кажется очевидным, но его строгое доказательство потребовало двух тысячелетий! Древние греки (включая Зенодора) догадывались, но не могли доказать. Лишь в XIX веке Якоб Штейнер предложил элегантное геометрическое доказательство методом симметризации.

📏 Изопериметрическое неравенство

Для любой замкнутой кривой длины \(L\), ограничивающей площадь \(A\): \[ L^2 \ge 4\pi A \] Равенство достигается только для окружности.

Проверка для круга:

\(L = 2\pi R\), \(A = \pi R^2\).

\(L^2 = 4\pi^2 R^2 = 4\pi \cdot \pi R^2 = 4\pi A\) — равенство выполняется.

Вывод: круг даёт максимальную площадь при заданном периметре.

🌍 Трёхмерный случай

В пространстве изопериметрическая задача звучит так: среди всех поверхностей с заданной площадью \(S\) найти ту, которая ограничивает максимальный объём \(V\).
Ответ: сфера.
Неравенство: \[ S^3 \ge 36\pi V^2 \] Равенство — для сферы.

⚡ Следствия для олимпиад

На олимпиадах часто встречаются задачи-следствия из изопериметрического неравенства:

1. Среди прямоугольников с данным периметром наибольшую площадь имеет квадрат.

Доказательство: \(xy \le \left(\frac{x+y}{2}\right)^2\). При фиксированном \(x+y\) произведение максимально, когда \(x=y\).

2. Среди прямоугольников с данной площадью наименьший периметр имеет квадрат.

Доказательство: \(x+y \ge 2\sqrt{xy}\). При фиксированном \(xy\) сумма минимальна, когда \(x=y\).

3. Среди всех цилиндров с заданным объёмом наименьшую площадь поверхности имеет цилиндр, у которого высота равна диаметру основания (осевое сечение — квадрат).

4. Вписанный в шар конус максимального объёма имеет объём \(\frac{8}{27}\) объёма шара.

✍️ Практика

Задача 1. Среди прямоугольников с периметром \(P=20\) найдите максимальную площадь.

Решение (пошагово):

Шаг 1. Пусть стороны прямоугольника — \(x\) и \(y\). Тогда периметр: \(2x+2y=20\) ⇒ \(x+y=10\).

Шаг 2. Площадь: \(S=xy\).

Шаг 3. По неравенству Коши: \(\frac{x+y}{2} \ge \sqrt{xy}\) ⇒ \(xy \le \left(\frac{x+y}{2}\right)^2\).

Шаг 4. Подставляем \(x+y=10\): \(S \le \left(\frac{10}{2}\right)^2 = 25\).

Шаг 5. Равенство достигается, когда \(x=y\).

Шаг 6. \(x=y=5\) (квадрат).

Ответ: максимальная площадь равна 25.

Задача 2. Среди прямоугольников с площадью \(S=16\) найдите наименьший периметр.

Решение (пошагово):

Шаг 1. Пусть стороны \(x\) и \(y\). \(xy=16\).

Шаг 2. Периметр: \(P=2(x+y)\).

Шаг 3. По неравенству Коши: \(\frac{x+y}{2} \ge \sqrt{xy}\) ⇒ \(x+y \ge 2\sqrt{xy}\).

Шаг 4. Подставляем \(xy=16\): \(x+y \ge 2\sqrt{16} = 8\).

Шаг 5. Периметр: \(P=2(x+y) \ge 16\).

Шаг 6. Равенство при \(x=y=4\) (квадрат).

Ответ: наименьший периметр равен 16.

Задача 3 (олимпиадная). Докажите, что среди всех треугольников с заданным периметром наибольшую площадь имеет равносторонний треугольник.

Решение (пошагово):

Шаг 1. По формуле Герона: \(S^2 = p(p-a)(p-b)(p-c)\), где \(p\) — полупериметр. При фиксированном периметре \(p\) постоянно.

Шаг 2. Обозначим: \(x=p-a\), \(y=p-b\), \(z=p-c\). Тогда \(x+y+z = 3p - (a+b+c) = 3p-2p = p\).

Шаг 3. Произведение \(xyz\) при фиксированной сумме \(x+y+z=p\) максимально, когда \(x=y=z\) (неравенство Коши для трёх чисел).

Шаг 4. \(x=y=z\) ⇔ \(p-a=p-b=p-c\) ⇔ \(a=b=c\).

Шаг 5. Значит, максимальная площадь — у равностороннего треугольника.

Ответ: равносторонний треугольник.
