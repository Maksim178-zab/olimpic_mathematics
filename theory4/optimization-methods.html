🎯 Что такое геометрическая оптимизация?

Геометрическая оптимизация — раздел математики, в котором ищут наилучшую (экстремальную) форму, размер или положение геометрического объекта при заданных ограничениях.

Примеры:

  • Максимальная площадь при фиксированном периметре
  • Минимальная длина дорог между точками
  • Наилучший обзор картины
  • Оптимальная форма резервуара
«В мире не происходит ничего, в чём бы не был виден смысл какого-нибудь максимума или минимума» — Леонард Эйлер

📐 Метод 1: сведение к функции одной переменной

Алгоритм:

Шаг 1. Ввести переменную (угол, длину отрезка, радиус).

Шаг 2. Выразить оптимизируемую величину (площадь, объём, расстояние) через эту переменную.

Шаг 3. Найти экстремум с помощью производной или неравенства.

Пример (классика). Окно имеет форму прямоугольника, завершённого сверху полукругом. Периметр окна равен \(P\). При каком радиусе полукруга площадь окна максимальна?

Решение (пошагово):

Шаг 1. Пусть радиус полукруга \(R\), высота прямоугольной части \(h\).

Шаг 2. Площадь: \(S = 2Rh + \frac{\pi R^2}{2}\).

Шаг 3. Периметр: \(P = 2h + 2R + \pi R\). (Нижняя сторона \(2R\), две боковые стороны \(2h\), дуга полукруга \(\pi R\).)

Шаг 4. Выражаем \(h\) из периметра: \(2h = P - 2R - \pi R\) ⇒ \(h = \frac{P - 2R - \pi R}{2}\).

Шаг 5. Подставляем в площадь:

\(S(R) = 2R \cdot \frac{P - 2R - \pi R}{2} + \frac{\pi R^2}{2} = R(P - 2R - \pi R) + \frac{\pi R^2}{2}\).

\(S(R) = PR - 2R^2 - \pi R^2 + \frac{\pi R^2}{2} = PR - 2R^2 - \frac{\pi R^2}{2}\).

Шаг 6. Находим производную: \(S'(R) = P - 4R - \pi R\).

Шаг 7. Приравниваем к нулю: \(P - 4R - \pi R = 0\) ⇒ \(P = R(4 + \pi)\) ⇒ \(R = \frac{P}{4+\pi}\).

Ответ: \(R = \frac{P}{4+\pi}\).

🔍 Метод 2: принцип отражения (кратчайший путь)

Принцип: Чтобы найти кратчайший путь от точки \(A\) до точки \(B\) с отражением от прямой, отражаем одну из точек симметрично и соединяем прямой.

Наглядное объяснение:

Представьте, что вы играете в бильярд. Шар должен удариться о бортик и попасть в лузу. Отражение точки относительно бортика превращает ломаную траекторию в прямую.

Путь \(A \to P \to B\), где \(P\) лежит на прямой \(l\), минимален, когда \(P\) лежит на отрезке \(A'B\), где \(A'\) — отражение \(A\) относительно \(l\).

Пример. Точки \(A\) и \(B\) находятся по одну сторону от прямой. Найдите на прямой точку \(P\), для которой \(AP + PB\) минимально.

Решение (пошагово):

Шаг 1. Отражаем точку \(A\) относительно прямой — получаем точку \(A'\).

Шаг 2. Соединяем \(A'\) и \(B\) прямой.

Шаг 3. Пересечение этой прямой с исходной прямой даёт точку \(P\).

Почему это работает? Для любой точки \(Q\) на прямой, \(AQ = A'Q\). Поэтому \(AQ + QB = A'Q + QB \ge A'B\) (по неравенству треугольника).

Минимум достигается, когда \(Q\) лежит на прямой \(A'B\).

Это классическая задача о кратчайшем пути с отражением (принцип Ферма).

📊 Метод 3: неравенства (Коши, Йенсен, изопериметрическое)

Идея: Олимпиадные задачи часто решаются без производной — через классические неравенства.

Основные инструменты:

  • Коши: \(\frac{x+y}{2} \ge \sqrt{xy}\) — для суммы и произведения
  • Йенсен: для выпуклых/вогнутых функций
  • Изопериметрическое: \(L^2 \ge 4\pi A\)

Пример. Доказать, что среди прямоугольников с заданным периметром максимальную площадь имеет квадрат.

Решение:

Шаг 1. Пусть стороны прямоугольника \(x\) и \(y\). Периметр \(P=2(x+y)\) фиксирован.

Шаг 2. Площадь \(S=xy\).

Шаг 3. По неравенству Коши: \(\frac{x+y}{2} \ge \sqrt{xy}\) ⇒ \(xy \le \left(\frac{x+y}{2}\right)^2\).

Шаг 4. При фиксированном \(x+y\) произведение максимально, когда \(x=y\).

Шаг 5. \(x=y\) ⇒ фигура — квадрат.

Вывод: Максимальная площадь у квадрата.

🔄 Метод 4: геометрические преобразования

Инструменты: повороты, симметрии, гомотетии, сдвиги.

Примеры:

  • Задача Ферма–Торричелли решается поворотом на \(60^\circ\).
  • Принцип симметризации Штейнера показывает, что невыпуклую или несимметричную фигуру можно «улучшить» без изменения периметра, но с ростом площади.
  • Принцип отражения — частный случай использования симметрии.

Почему поворот помогает?

Поворот позволяет «выпрямить» ломаную линию в прямую, после чего задача становится тривиальной — нужно просто найти расстояние между двумя точками.

🌍 Почему это важно (наглядные примеры из жизни)

Мыльные пузыри принимают сферическую форму — это минимизация площади поверхности при фиксированном объёме (трёхмерная изопериметрическая задача).

Пчёлы строят соты шестиугольной формы — это оптимальное заполнение плоскости, дающее максимальную площадь при минимальном периметре ячеек.

Капли воды в невесомости становятся шарами — та же причина.

Геометрическая оптимизация — мост между чистой математикой и реальным миром. Освоив её, вы научитесь видеть скрытую гармонию форм в природе и технике.

✍️ Практика

Задача 1 (метод отражения). Точка \(A\) находится на расстоянии 3 см от прямой, точка \(B\) — на расстоянии 5 см. Расстояние между проекциями точек на прямую — 10 см. Найдите длину кратчайшего пути \(A \to P \to B\), где \(P\) лежит на прямой.

Решение (пошагово):

Шаг 1. Отражаем точку \(A\) относительно прямой. Она оказывается на расстоянии 3 см по другую сторону прямой.

Шаг 2. Расстояние между отражённой точкой \(A'\) и точкой \(B\) вычисляется по теореме Пифагора.

Шаг 3. По горизонтали: 10 см. По вертикали: \(3+5=8\) см.

Шаг 4. \(A'B = \sqrt{10^2 + 8^2} = \sqrt{100+64} = \sqrt{164} = 2\sqrt{41}\).

Ответ: \(2\sqrt{41}\) см.

Задача 2 (функция). Точка \(A(a,0)\) лежит на оси \(x\), точка \(B(0,b)\) — на оси \(y\). Найдите минимальную длину отрезка, соединяющего \(A\) и \(B\), если \(a+b=10\) (a,b > 0).

Решение (пошагово):

Шаг 1. Длина отрезка: \(L = \sqrt{a^2 + b^2}\).

Шаг 2. По условию \(b=10-a\).

Шаг 3. \(L^2 = a^2 + (10-a)^2 = a^2 + 100 - 20a + a^2 = 2a^2 - 20a + 100\).

Шаг 4. Квадратичная функция \(L^2(a) = 2a^2 - 20a + 100\).

Шаг 5. Вершина параболы: \(a = -\frac{-20}{2\cdot2} = \frac{20}{4} = 5\).

Шаг 6. Тогда \(b=5\).

Шаг 7. Минимальная длина: \(L = \sqrt{5^2+5^2} = \sqrt{50} = 5\sqrt{2}\).

Ответ: \(5\sqrt{2}\).

Задача 3 (вписанный цилиндр). В конус с радиусом основания \(R\) и высотой \(H\) вписан цилиндр. Найдите его максимальный объём.

Решение (пошагово):

Шаг 1. Рассмотрим осевое сечение — равнобедренный треугольник с вписанным прямоугольником.

Шаг 2. Из подобия треугольников: \(\frac{r}{R} = \frac{H-h}{H}\) ⇒ \(r = R\left(1 - \frac{h}{H}\right)\).

Шаг 3. Объём цилиндра: \(V(h) = \pi r^2 h = \pi R^2 \left(1 - \frac{h}{H}\right)^2 h\).

Шаг 4. Найдём максимум функции \(f(h) = \left(1 - \frac{h}{H}\right)^2 h\).

Шаг 5. Производная: \(f'(h) = \left(1 - \frac{h}{H}\right)\left(1 - \frac{3h}{H}\right)\).

Шаг 6. Приравниваем к нулю: \(h=H\) (не подходит, цилиндр нулевой) или \(h = \frac{H}{3}\).

Шаг 7. Тогда \(r = R\left(1 - \frac{1}{3}\right) = \frac{2R}{3}\).

Шаг 8. \(V_{\max} = \pi \left(\frac{2R}{3}\right)^2 \cdot \frac{H}{3} = \pi \cdot \frac{4R^2}{9} \cdot \frac{H}{3} = \frac{4\pi R^2 H}{27}\).

Ответ: \(\frac{4}{27}\pi R^2 H\).
