📏 Часть 1. Изопериметрические задачи (без производной)

Задача 1. Периметр прямоугольника равен 20 см. Найдите его наибольшую возможную площадь.

Подробное решение:

Шаг 1. Вводим переменные. Пусть стороны прямоугольника равны \(x\) и \(y\) (в сантиметрах). Это стандартный приём: любая задача про прямоугольник начинается с обозначения его сторон через буквы.

Шаг 2. Записываем условие. Периметр — это сумма всех сторон: \(P = x + y + x + y = 2x + 2y = 2(x+y)\). По условию \(P = 20\), значит: \(2(x+y) = 20 \Rightarrow x+y = 10\).

Шаг 3. Что нужно найти? Площадь прямоугольника: \(S = xy\). Нам нужно найти наибольшее значение произведения \(xy\) при условии, что \(x+y=10\) и \(x>0, y>0\).

Шаг 4. Каким инструментом воспользоваться? Здесь работает неравенство Коши (о среднем арифметическом и среднем геометрическом). Оно гласит: для любых неотрицательных чисел \(a\) и \(b\) выполняется \(\frac{a+b}{2} \ge \sqrt{ab}\). Почему оно подходит? Потому что в нём есть и сумма (\(a+b\)), и произведение (\(ab\)) — именно то, что нам нужно.

Шаг 5. Применяем неравенство Коши к числам \(x\) и \(y\): \(\frac{x+y}{2} \ge \sqrt{xy}\). Подставляем \(x+y=10\): \(\frac{10}{2} \ge \sqrt{xy} \Rightarrow 5 \ge \sqrt{S}\). Возводим обе части в квадрат (обе неотрицательны, можно): \(25 \ge S\).

Шаг 6. Когда достигается равенство? В неравенстве Коши равенство достигается тогда и только тогда, когда \(x = y\). Это ключевой момент: если \(x=y\), то \(x+x=10 \Rightarrow x=5\), \(y=5\). Тогда площадь \(S = 5 \cdot 5 = 25\).

Вывод. Максимальная площадь равна 25 см², и она достигается для квадрата со стороной 5 см.

Ответ: 25 см².

Задача 2. Площадь прямоугольника равна 36 см². Найдите его наименьший возможный периметр.

Подробное решение:

Шаг 1. Пусть стороны \(x\) и \(y\). Площадь: \(xy = 36\).

Шаг 2. Что нужно найти? Периметр \(P = 2(x+y)\). Чтобы минимизировать \(P\), нужно минимизировать \(x+y\).

Шаг 3. Применяем Коши: \(\frac{x+y}{2} \ge \sqrt{xy} = \sqrt{36} = 6\). Откуда берётся корень? Из условия \(xy=36\). Умножаем на 2: \(x+y \ge 12\).

Шаг 4. Равенство достигается при \(x = y = 6\). Тогда периметр \(P = 2(6+6) = 24\).

Шаг 5. Почему это минимум? Мы доказали, что \(x+y\) не может быть меньше 12, значит, периметр не может быть меньше 24.

Ответ: 24 см.

Задача 3. Среди всех прямоугольников с диагональю \(d\) найти наибольшую площадь.

Подробное решение:

Шаг 1. Пусть стороны \(x\) и \(y\). Диагональ связана с ними теоремой Пифагора: \(x^2 + y^2 = d^2\).

Шаг 2. Что нужно найти? Максимум \(S = xy\). Нам нужно связать \(xy\) с \(x^2+y^2\). Вспоминаем тождество: \((x-y)^2 = x^2 - 2xy + y^2 \ge 0\). Отсюда: \(x^2 + y^2 \ge 2xy\). Это неравенство верно для любых чисел. Подставляем \(x^2+y^2 = d^2\): \(d^2 \ge 2xy \Rightarrow xy \le \frac{d^2}{2}\).

Шаг 3. Равенство достигается, когда \((x-y)^2 = 0\), то есть \(x=y\). Тогда прямоугольник — квадрат, и его диагональ \(d = x\sqrt{2}\), значит, \(x = y = \frac{d}{\sqrt{2}}\). Площадь: \(S = \frac{d^2}{2}\).

Ответ: \(\frac{d^2}{2}\).

Задача 4. Среди всех прямоугольников, вписанных в окружность радиуса \(R\), найти наибольшую площадь.

Подробное решение:

Шаг 1. Любой прямоугольник, вписанный в окружность, имеет диагональ, равную диаметру окружности. Почему? Потому что угол, опирающийся на диаметр, — прямой. Значит, диагональ прямоугольника — это диаметр окружности. \(d = 2R\).

Шаг 2. Используем результат задачи 3. Мы уже доказали, что для прямоугольника с диагональю \(d\) максимальная площадь равна \(\frac{d^2}{2}\). Подставляем \(d = 2R\): \(S_{\max} = \frac{(2R)^2}{2} = \frac{4R^2}{2} = 2R^2\).

Шаг 3. Когда достигается? При \(x=y\), то есть когда прямоугольник является квадратом. Квадрат, вписанный в окружность, имеет сторону \(R\sqrt{2}\).

Ответ: \(2R^2\).

Задача 5. Среди всех треугольников с данным периметром \(P\) найти наибольшую площадь.

Подробное решение:

Шаг 1. Формула Герона. Площадь треугольника со сторонами \(a, b, c\) и полупериметром \(p = \frac{P}{2}\) равна: \(S = \sqrt{p(p-a)(p-b)(p-c)}\). Эта формула берётся из геометрии (доказывается через высоты). Мы её используем как готовый инструмент.

Шаг 2. Что в ней фиксировано? \(p\) — константа (половина периметра). Значит, чтобы максимизировать \(S\), нужно максимизировать произведение \((p-a)(p-b)(p-c)\).

Шаг 3. Замечаем сумму: \((p-a)+(p-b)+(p-c) = 3p - (a+b+c) = 3p - 2p = p\). Сумма трёх чисел \((p-a), (p-b), (p-c)\) фиксирована и равна \(p\).

Шаг 4. Применяем неравенство Коши для трёх чисел. Для неотрицательных чисел \(u, v, w\): \(\frac{u+v+w}{3} \ge \sqrt[3]{uvw}\). Значит, \(uvw \le \left(\frac{u+v+w}{3}\right)^3\). Подставляем \(u=p-a, v=p-b, w=p-c\): \((p-a)(p-b)(p-c) \le \left(\frac{p}{3}\right)^3 = \frac{p^3}{27}\).

Шаг 5. Подставляем в формулу Герона: \(S \le \sqrt{p \cdot \frac{p^3}{27}} = \sqrt{\frac{p^4}{27}} = \frac{p^2}{3\sqrt{3}} = \frac{P^2}{36\sqrt{3}} = \frac{P^2\sqrt{3}}{36}\).

Шаг 6. Равенство достигается, когда \(p-a = p-b = p-c\), то есть \(a=b=c\). Треугольник равносторонний.

Ответ: \(\frac{P^2\sqrt{3}}{36}\).

Задача 6. Среди всех треугольников, вписанных в окружность радиуса \(R\), найти наибольшую площадь.

Подробное решение:

Шаг 1. Формула площади через радиус описанной окружности. Для любого треугольника: \(S = \frac{abc}{4R}\). Это известная формула, она следует из \(S = \frac{1}{2}ab\sin C\) и \(c = 2R\sin C\).

Шаг 2. Что нужно сделать? При фиксированном \(R\) максимизировать \(S\) — значит максимизировать произведение \(abc\).

Шаг 3. Какое ограничение на \(a, b, c\)? Сумма сторон не фиксирована, но есть неравенство: для любого треугольника \(a+b+c \le 3\sqrt{3}R\) (это следует из того, что сумма синусов углов ≤ \(3\sqrt{3}/2\)).

Шаг 4. Более простой путь: известный факт. Среди всех треугольников, вписанных в данную окружность, равносторонний имеет наибольшую площадь. Этот факт часто принимается без доказательства на олимпиадах. Но мы можем его обосновать: если зафиксировать одну сторону, площадь максимальна, когда треугольник равнобедренный (высота максимальна). Затем варьируя основание, получаем равносторонний.

Шаг 5. Вычисляем площадь равностороннего треугольника. Сторона равностороннего треугольника, вписанного в окружность радиуса \(R\): \(a = R\sqrt{3}\). (Почему? Центр описанной окружности совпадает с центром треугольника, расстояние от центра до вершины — \(R\), а сторона равна \(R\sqrt{3}\).) Площадь: \(S = \frac{\sqrt{3}}{4} a^2 = \frac{\sqrt{3}}{4} \cdot 3R^2 = \frac{3\sqrt{3}}{4}R^2\).

Ответ: \(\frac{3\sqrt{3}}{4}R^2\).

Задача 7. Окно имеет форму прямоугольника, завершённого сверху полукругом. Периметр окна равен \(P\). Найдите радиус полукруга, при котором площадь окна максимальна.

Подробное решение:

Шаг 1. Вводим переменные. \(R\) — радиус полукруга. Тогда ширина прямоугольника равна \(2R\) (так как полукруг опирается на ширину). \(h\) — высота прямоугольной части.

Шаг 2. Записываем периметр. Периметр окна состоит из: двух вертикальных сторон прямоугольника: \(h + h = 2h\); нижней стороны прямоугольника: \(2R\); полуокружности: длина полуокружности равна \(\pi R\) (половина от \(2\pi R\)). Итого: \(P = 2h + 2R + \pi R\).

Шаг 3. Выражаем \(h\) через \(R\): \(2h = P - 2R - \pi R \Rightarrow h = \frac{P - R(2+\pi)}{2}\).

Шаг 4. Записываем площадь. Площадь окна = площадь прямоугольника + площадь полукруга: \(S = 2R \cdot h + \frac{1}{2}\pi R^2\). Подставляем \(h\): \(S = 2R \cdot \frac{P - R(2+\pi)}{2} + \frac{1}{2}\pi R^2 = R(P - R(2+\pi)) + \frac{\pi}{2}R^2\). Раскрываем скобки: \(S = PR - 2R^2 - \pi R^2 + \frac{\pi}{2}R^2 = PR - \left(2 + \frac{\pi}{2}\right)R^2\).

Шаг 5. Что мы получили? Квадратичную функцию от \(R\): \(S(R) = -\left(2 + \frac{\pi}{2}\right)R^2 + PR\). Коэффициент при \(R^2\) отрицательный, значит, это парабола ветвями вниз, у неё есть максимум.

Шаг 6. Находим вершину параболы. Для функции \(ax^2 + bx + c\) вершина находится в точке \(x = -\frac{b}{2a}\). Здесь \(a = -\left(2 + \frac{\pi}{2}\right)\), \(b = P\). \(R = -\frac{P}{2\left(-\left(2 + \frac{\pi}{2}\right)\right)} = \frac{P}{2\left(2 + \frac{\pi}{2}\right)} = \frac{P}{4 + \pi}\).

Шаг 7. Почему это максимум? Парабола ветвями вниз, вершина даёт максимум.

Ответ: \(R = \frac{P}{4+\pi}\).

Задача 8. Среди всех цилиндров с заданным объёмом \(V\) найти цилиндр с наименьшей полной поверхностью.

Подробное решение:

Шаг 1. Вводим переменные. \(r\) — радиус основания, \(h\) — высота.

Шаг 2. Объём: \(V = \pi r^2 h \Rightarrow h = \frac{V}{\pi r^2}\).

Шаг 3. Площадь полной поверхности: \(S = 2\pi r^2 + 2\pi r h\). Подставляем \(h\): \(S = 2\pi r^2 + 2\pi r \cdot \frac{V}{\pi r^2} = 2\pi r^2 + \frac{2V}{r}\).

Шаг 4. Как минимизировать \(S\)? У нас три слагаемых (если разбить \(\frac{2V}{r}\) на два): \(S = 2\pi r^2 + \frac{V}{r} + \frac{V}{r}\). Все три слагаемые положительны. Применяем неравенство Коши для трёх чисел: \(\frac{a+b+c}{3} \ge \sqrt[3]{abc}\). Пусть \(a = 2\pi r^2\), \(b = \frac{V}{r}\), \(c = \frac{V}{r}\). Тогда: \(\frac{S}{3} \ge \sqrt[3]{2\pi r^2 \cdot \frac{V}{r} \cdot \frac{V}{r}} = \sqrt[3]{2\pi V^2}\). Следовательно: \(S \ge 3\sqrt[3]{2\pi V^2}\).

Шаг 5. Равенство. В неравенстве Коши равенство достигается, когда \(a=b=c\), то есть: \(2\pi r^2 = \frac{V}{r}\). Умножаем на \(r\): \(2\pi r^3 = V\). Тогда \(h = \frac{V}{\pi r^2} = \frac{2\pi r^3}{\pi r^2} = 2r\). Значит, высота равна диаметру основания.

Ответ: \(h = 2r\) (осевое сечение — квадрат).

Задача 9. Среди всех конусов с заданной образующей \(l\) найти конус с наибольшим объёмом.

Подробное решение:

Шаг 1. Вводим переменные. \(r\) — радиус основания, \(h\) — высота. Образующая \(l\) связана с ними: \(l^2 = r^2 + h^2\).

Шаг 2. Объём конуса: \(V = \frac{1}{3}\pi r^2 h\). Из \(r^2 = l^2 - h^2\) получаем: \(V(h) = \frac{1}{3}\pi (l^2 - h^2)h = \frac{1}{3}\pi (l^2 h - h^3)\).

Шаг 3. Нужно максимизировать \(f(h) = l^2 h - h^3\), где \(0 < h < l\).

Шаг 4. Найдём производную: \(f'(h) = l^2 - 3h^2\). Приравниваем к нулю: \(l^2 - 3h^2 = 0 \Rightarrow h = \frac{l}{\sqrt{3}}\) (положительный корень).

Шаг 5. Проверяем знаки: при \(h < \frac{l}{\sqrt{3}}\) производная положительна (функция растёт), при \(h > \frac{l}{\sqrt{3}}\) производная отрицательна (функция убывает). Значит, \(h = \frac{l}{\sqrt{3}}\) — точка максимума.

Шаг 6. Тогда \(r^2 = l^2 - \frac{l^2}{3} = \frac{2l^2}{3}\), значит \(r = l\sqrt{\frac{2}{3}}\).

Шаг 7. Объём: \(V_{\max} = \frac{1}{3}\pi \cdot \frac{2l^2}{3} \cdot \frac{l}{\sqrt{3}} = \frac{2\pi l^3}{9\sqrt{3}}\).

Ответ: \(h = \frac{l}{\sqrt{3}}\), \(r = l\sqrt{\frac{2}{3}}\), \(V_{\max} = \frac{2\pi l^3}{9\sqrt{3}}\).

Задача 10 (Дидона). Верёвка длины \(L\) огораживает участок вдоль прямой стены. Какую форму должен иметь участок, чтобы площадь была максимальной?

Подробное решение:

Шаг 1. Понимание задачи. Стена — это прямая линия. Верёвка используется только для трёх сторон участка: две боковые и одна дальняя от стены. Четвёртая сторона — стена.

Шаг 2. Изопериметрический принцип. Изопериметрическая задача на плоскости говорит: среди всех замкнутых кривых данной длины максимальную площадь имеет окружность. Если часть границы «бесплатна» (стена), то оптимальная фигура — полукруг, опирающийся на стену как на диаметр.

Шаг 3. Почему полукруг? Потому что если отразить полукруг относительно стены, получится полный круг, и верёвка длины \(L\) станет половиной окружности этого круга. Площадь полукруга будет половиной площади круга.

Шаг 4. Вычисления. Длина верёвки равна полуокружности: \(L = \pi R \Rightarrow R = \frac{L}{\pi}\). Площадь полукруга: \(S = \frac{1}{2}\pi R^2 = \frac{1}{2}\pi \cdot \frac{L^2}{\pi^2} = \frac{L^2}{2\pi}\).

Шаг 5. Почему полукруг? Любая другая форма даст меньшую площадь при той же длине верёвки — это следует из изопериметрического неравенства.

Ответ: полукруг радиуса \(L/\pi\), площадь \(L^2/(2\pi)\).

🏔️ Часть 2. Задача Ферма–Торричелли

Задача 11. Дан треугольник \(ABC\) с углами меньше 120°. Найдите точку, сумма расстояний от которой до вершин минимальна.

Подробное решение:

Шаг 1. Что такое точка Ферма–Торричелли? Это точка \(T\), для которой сумма расстояний до вершин \(TA + TB + TC\) минимальна.

Шаг 2. Ключевой геометрический факт. Если все углы треугольника меньше 120°, то точка минимума — это единственная точка, из которой все стороны треугольника видны под углом 120°: \(\angle ATB = \angle BTC = \angle CTA = 120^\circ\).

Шаг 3. Откуда берётся угол 120°? Это следует из метода поворота. Повернём треугольник \(BTC\) вокруг точки \(B\) на 60°. Тогда отрезок \(TC\) перейдёт в \(TC'\), а \(BT\) — в \(BT'\). Точки \(A, T, T', C'\) окажутся на одной прямой тогда и только тогда, когда углы между \(TA, TB, TC\) равны 120°. Длина \(AT + BT + CT\) становится длиной прямой \(AC'\), что минимально.

Шаг 4. Вывод. Точка минимума — это точка, из которой отрезки к вершинам образуют углы 120°.

Ответ: точка Ферма–Торричелли (углы 120°).

Задача 12. В треугольнике \(ABC\) угол \(A = 130^\circ\). Где находится точка минимума суммы расстояний до вершин?

Подробное решение:

Шаг 1. Правило для тупого угла. Если один из углов треугольника больше или равен 120°, точка минимума суммы расстояний совпадает с вершиной этого тупого угла.

Шаг 2. Почему? Рассмотрим сумму \(MA + MB + MC\). Если \(M\) совпадает с вершиной \(A\), то сумма равна \(AB + AC\). Любая другая точка \(M\) даст \(MA + MB + MC > AB + AC\) по неравенству треугольника (ломаная длиннее прямой).

Шаг 3. В нашей задаче угол \(A = 130^\circ > 120^\circ\), значит, минимум в точке \(A\).

Ответ: точка \(A\).

Задача 13. Дан равносторонний треугольник со стороной \(a\). Найдите сумму расстояний от точки Ферма до вершин.

Подробное решение:

Шаг 1. Где находится точка Ферма в равностороннем треугольнике? В равностороннем треугольнике точка Ферма совпадает с центром (он же центр описанной окружности, центр вписанной окружности, точка пересечения медиан). Все углы между отрезками к вершинам — 120°.

Шаг 2. Расстояние от центра до вершины. Центр равностороннего треугольника находится на расстоянии \(R = \frac{a}{\sqrt{3}}\) от каждой вершины (это радиус описанной окружности). Откуда берётся эта формула? В равностороннем треугольнике высота равна \(\frac{a\sqrt{3}}{2}\), центр делит её в отношении 2:1 от вершины, значит, расстояние от вершины до центра равно \(\frac{2}{3} \cdot \frac{a\sqrt{3}}{2} = \frac{a}{\sqrt{3}}\).

Шаг 3. Сумма расстояний. Так как расстояния от точки Ферма до всех трёх вершин равны, сумма: \(S = 3 \cdot \frac{a}{\sqrt{3}} = a\sqrt{3}\).

Ответ: \(a\sqrt{3}\).

Задача 14. Дан квадрат \(ABCD\) со стороной \(a\). Найдите точку, сумма расстояний от которой до вершин минимальна.

Подробное решение:

Шаг 1. Симметрия. Квадрат симметричен относительно центра. Интуитивно ясно, что минимум суммы расстояний до всех вершин должен достигаться в центре симметрии.

Шаг 2. Проверим. Пусть \(M\) — центр квадрата. Расстояние от центра до каждой вершины равно половине диагонали: \(\frac{a\sqrt{2}}{2} = \frac{a}{\sqrt{2}}\).

Шаг 3. Сумма: \(4 \cdot \frac{a}{\sqrt{2}} = 2a\sqrt{2}\).

Шаг 4. Почему не может быть меньше? Для любой точки \(M\) сумма расстояний до противоположных вершин не меньше диагонали (по неравенству треугольника). Сумма двух таких пар даёт \(2a\sqrt{2}\). Равенство достигается, когда точка лежит на обеих диагоналях, то есть в центре.

Ответ: центр квадрата, сумма \(2a\sqrt{2}\).

Задача 15. Даны две точки \(A\) и \(B\) и прямая \(l\). Найдите на прямой \(l\) точку \(M\) такую, что \(MA + MB\) минимально.

Подробное решение:

Шаг 1. Классический метод отражения. Отражаем точку \(A\) симметрично относительно прямой \(l\). Получаем точку \(A'\). Почему это помогает? Потому что для любой точки \(M\) на прямой \(l\) расстояние \(MA = MA'\) (по определению симметрии, прямая — серединный перпендикуляр к отрезку \(AA'\)).

Шаг 2. Преобразуем сумму: \(MA + MB = MA' + MB\).

Шаг 3. Неравенство треугольника. Для любых двух точек \(A'\) и \(B\) длина ломаной \(MA' + MB\) не меньше длины отрезка \(A'B\). Равенство достигается, когда \(M\) лежит на отрезке \(A'B\).

Шаг 4. Построение. Точка \(M\) — это пересечение прямой \(l\) с отрезком \(A'B\).

Ответ: пересечение \(l\) и \(A'B\).

Задача 16. Даны две точки \(A\) и \(B\) по одну сторону от прямой \(l\). Найдите на \(l\) точку \(M\), минимизирующую \(MA+MB\).

Подробное решение:

Шаг 1. Та же техника. Отражаем \(A\) в \(A'\) (симметрично относительно \(l\)). Тогда \(MA = MA'\).

Шаг 2. Минимизируем \(MA' + MB\). Это сумма расстояний от \(M\) до двух точек \(A'\) и \(B\). Минимум достигается, когда \(M\) лежит на отрезке \(A'B\).

Шаг 3. Результат. Точка \(M\) — пересечение \(A'B\) с \(l\).

Ответ: пересечение отрезка \(A'B\) с прямой \(l\).

Задача 17. Даны три точки \(A, B, C\). Найдите точку \(M\), минимизирующую \(MA+MB+MC\), если все углы треугольника \(<120^\circ\). Опишите построение.

Подробное решение:

Шаг 1. Построение Штейнера. На каждой стороне треугольника \(ABC\) наружу строим равносторонний треугольник:

  • на \(AB\) строим \(ABD\);
  • на \(BC\) строим \(BCE\);
  • на \(CA\) строим \(CAF\).

Шаг 2. Проводим отрезки. Соединяем свободные вершины этих равносторонних треугольников с противоположными вершинами исходного треугольника:

  • \(D\) с \(C\);
  • \(E\) с \(A\);
  • \(F\) с \(B\).

Шаг 3. Теорема. Эти три отрезка пересекаются в одной точке. Эта точка и есть точка Ферма–Торричелли.

Шаг 4. Почему это работает? Метод основан на повороте на 60°. Поворот одной из частей треугольника выстраивает сумму расстояний в прямую линию, и пересечение отрезков даёт точку минимума.

Ответ: точка пересечения отрезков, соединяющих вершины равносторонних треугольников с противоположными вершинами исходного.

Задача 18. В треугольнике со сторонами \(a, b, c\) точка Ферма даёт сумму расстояний \(S\). Верно ли, что \(S < a+b+c\)?

Подробное решение:

Шаг 1. Неравенство треугольника. Для любой точки \(M\) внутри треугольника: \(MA < AB + AC = c+b\). Аналогично: \(MB < a+c\), \(MC < a+b\).

Шаг 2. Складываем: \(S = MA+MB+MC < (b+c) + (a+c) + (a+b) = 2(a+b+c)\). Это слишком слабое неравенство.

Шаг 3. Но для точки Ферма можно доказать более сильное утверждение. Из построения Штейнера следует, что \(S\) равно длине отрезка, соединяющего две вершины равносторонних треугольников. Эта длина меньше \(a+b+c\).

Шаг 4. Интуитивно: точка Ферма лежит внутри треугольника, и сумма расстояний до вершин меньше суммы двух сторон, потому что ломаная \(A \to T \to B\) короче ломаной \(A \to C \to B\) при определённых условиях. Для точки Ферма это всегда верно.

Ответ: да, \(S < a+b+c\).

Задача 19. Точка Ферма для треугольника \(ABC\) находится внутри треугольника. Что можно сказать об углах треугольника?

Подробное решение:

Шаг 1. Критерий. Точка Ферма лежит внутри треугольника тогда и только тогда, когда все углы треугольника меньше 120°.

Шаг 2. Почему? Если угол \(\ge 120^\circ\), минимум достигается в вершине этого угла (как в задаче 12). Если все углы \(<120^\circ\), точка минимума — внутренняя точка, из которой все стороны видны под 120°.

Ответ: все углы треугольника < 120°.

Задача 20. Дан отрезок \(AB\) длины \(l\). Найдите на перпендикуляре к \(AB\), восставленном в середине отрезка, точку \(M\), для которой \(MA + MB\) минимально.

Подробное решение:

Шаг 1. Понимание условия. Перпендикуляр к \(AB\) в середине — это прямая, проходящая через середину \(AB\) и перпендикулярная ему. Любая точка на этом перпендикуляре равноудалена от \(A\) и \(B\) (по свойству серединного перпендикуляра).

Шаг 2. Сумма расстояний. Для любой точки \(M\) на этом перпендикуляре: \(MA = MB\). Значит, \(MA + MB = 2MA\).

Шаг 3. Минимум. Чтобы минимизировать \(2MA\), нужно минимизировать \(MA\). Расстояние от точки на перпендикуляре до точки \(A\) минимально, когда точка находится в основании перпендикуляра — то есть в середине отрезка \(AB\).

Шаг 4. Значение минимума. В середине отрезка \(MA = \frac{l}{2}\), сумма \(= l\).

Ответ: середина отрезка \(AB\).

Задача 21. Из всех треугольников с заданным основанием \(a\) и углом при вершине \(\alpha\) найти максимальную площадь.

Подробное решение:

Шаг 1. Что дано? Основание \(a\) фиксировано. Угол при вершине \(\alpha\) (угол между боковыми сторонами) фиксирован.

Шаг 2. Высота и площадь. Площадь \(S = \frac{1}{2} a h\). Чтобы максимизировать площадь, нужно максимизировать высоту \(h\).

Шаг 3. Где высота максимальна? При фиксированном основании и фиксированном угле при вершине, высота максимальна, когда треугольник равнобедренный. Это следует из симметрии: при отклонении вершины от середины основания высота уменьшается.

Шаг 4. Вычисляем высоту для равнобедренного треугольника. В равнобедренном треугольнике высота, опущенная на основание, делит угол при вершине пополам. В прямоугольном треугольнике с гипотенузой (боковой стороной) и углом \(\alpha/2\) при вершине: \(\operatorname{tg}\frac{\alpha}{2} = \frac{a/2}{h} \Rightarrow h = \frac{a}{2\operatorname{tg}(\alpha/2)} = \frac{a}{2} \operatorname{ctg}\frac{\alpha}{2}\).

Шаг 5. Площадь: \(S = \frac{1}{2} a h = \frac{a^2}{4} \operatorname{ctg}\frac{\alpha}{2}\).

Ответ: равнобедренный треугольник, \(S = \frac{a^2}{4}\operatorname{ctg}\frac{\alpha}{2}\).

Задача 22. На эллипсе \(x^2/a^2 + y^2/b^2 = 1\) найти точки, ближайшие к центру.

Подробное решение:

Шаг 1. Расстояние от центра. Расстояние от точки \((x, y)\) до центра \((0, 0)\): \(r = \sqrt{x^2+y^2}\).

Шаг 2. На эллипсе. Минимальное значение \(r\) достигается в точках, где \(y=0\) (на концах малой оси) или \(x=0\) (на концах большой оси)? Нужно сравнить.

Шаг 3. На эллипсе \(x^2/a^2 + y^2/b^2 = 1\). Чтобы минимизировать \(x^2+y^2\), заметим, что при фиксированном уравнении эллипса сумма квадратов минимальна на концах малой оси, потому что \(b \le a\) (если \(a \ge b\)). Точки \((0, \pm b)\) дают \(r = b\).

Шаг 4. Максимум. Точки \((\pm a, 0)\) дают \(r = a\).

Ответ: \((0, \pm b)\).

Задача 23. На параболе \(y = x^2\) найти точку, ближайшую к точке \((0, 2)\).

Подробное решение:

Шаг 1. Расстояние. Точка на параболе имеет координаты \((x, x^2)\). Расстояние до \((0, 2)\): \(d^2 = (x-0)^2 + (x^2-2)^2 = x^2 + (x^2-2)^2\).

Шаг 2. Раскрываем скобки: \(d^2 = x^2 + x^4 - 4x^2 + 4 = x^4 - 3x^2 + 4\).

Шаг 3. Выделяем полный квадрат. \(x^4 - 3x^2 + 4 = (x^2)^2 - 2 \cdot \frac{3}{2} x^2 + \left(\frac{3}{2}\right)^2 + 4 - \left(\frac{3}{2}\right)^2\) = \(\left(x^2 - \frac{3}{2}\right)^2 + 4 - \frac{9}{4} = \left(x^2 - \frac{3}{2}\right)^2 + \frac{7}{4}\).

Шаг 4. Минимум. Квадрат всегда неотрицателен. Минимум достигается, когда квадрат равен нулю: \(x^2 - \frac{3}{2} = 0 \Rightarrow x = \pm\sqrt{\frac{3}{2}}\).

Шаг 5. Точки на параболе: \(y = x^2 = \frac{3}{2}\).

Ответ: \(\left(\pm\sqrt{\frac{3}{2}}, \frac{3}{2}\right)\).

Задача 24. На окружности \(x^2 + y^2 = R^2\) найти точку, ближайшую к точке \(A(a, b)\) (вне окружности).

Подробное решение:

Шаг 1. Геометрический факт. Ближайшая к точке \(A\) точка на окружности лежит на отрезке, соединяющем \(A\) с центром окружности. Это следует из того, что расстояние от \(A\) до любой точки окружности минимально, когда направление совпадает с радиусом.

Шаг 2. Координаты. Вектор от центра к точке \(A\): \((a, b)\). Его длина: \(r = \sqrt{a^2+b^2}\). Точка на окружности в том же направлении имеет координаты: \(\left(\frac{Ra}{r}, \frac{Rb}{r}\right) = \left(\frac{Ra}{\sqrt{a^2+b^2}}, \frac{Rb}{\sqrt{a^2+b^2}}\right)\).

Ответ: \(\left(\frac{Ra}{\sqrt{a^2+b^2}}, \frac{Rb}{\sqrt{a^2+b^2}}\right)\).

Задача 25. На прямой \(y = 2x + 1\) найти точку, ближайшую к началу координат.

Подробное решение:

Шаг 1. Точка на прямой. Любая точка на прямой имеет вид \((x, 2x+1)\).

Шаг 2. Расстояние до начала: \(d^2 = x^2 + (2x+1)^2 = x^2 + 4x^2 + 4x + 1 = 5x^2 + 4x + 1\).

Шаг 3. Выделяем квадрат: \(5x^2 + 4x + 1 = 5\left(x^2 + \frac{4}{5}x\right) + 1 = 5\left(\left(x + \frac{2}{5}\right)^2 - \frac{4}{25}\right) + 1\) = \(5\left(x + \frac{2}{5}\right)^2 - \frac{4}{5} + 1 = 5\left(x + \frac{2}{5}\right)^2 + \frac{1}{5}\).

Шаг 4. Минимум. При \(x = -\frac{2}{5}\) квадрат равен нулю. Тогда \(y = 2\left(-\frac{2}{5}\right) + 1 = -\frac{4}{5} + 1 = \frac{1}{5}\).

Ответ: \(\left(-\frac{2}{5}, \frac{1}{5}\right)\).

Задача 26. Найти наименьшее расстояние между параболой \(y = x^2\) и прямой \(y = x + 2\).

Подробное решение:

Шаг 1. Расстояние от точки до прямой. Точка на параболе: \((x, x^2)\). Расстояние от неё до прямой \(y = x + 2\) (перепишем: \(x - y + 2 = 0\)): \(d = \frac{|x - x^2 + 2|}{\sqrt{1^2 + (-1)^2}} = \frac{|x^2 - x - 2|}{\sqrt{2}}\).

Шаг 2. Минимум модуля. Нужно минимизировать \(|x^2 - x - 2|\). Это выражение равно нулю при \(x = 2\) или \(x = -1\) (корни квадратного трёхчлена). В этих точках расстояние равно 0.

Шаг 3. Что это значит? Парабола пересекает прямую. Значит, минимальное расстояние между ними равно 0.

Ответ: 0 (пересекаются).

Задача 27. Среди всех прямоугольников, вписанных в равнобедренный треугольник с основанием \(a\) и высотой \(h\) (основание прямоугольника лежит на основании треугольника), найти максимальную площадь.

Подробное решение:

Шаг 1. Чертёж. Основание треугольника лежит на оси \(x\). Вершина треугольника в точке \((0, h)\), основание от \((-a/2, 0)\) до \((a/2, 0)\).

Шаг 2. Прямоугольник. Пусть его ширина \(2x\) (от \(-x\) до \(x\)), высота \(y\). Верхние углы прямоугольника лежат на боковых сторонах треугольника.

Шаг 3. Находим связь \(x\) и \(y\). Боковая сторона треугольника — прямая из \((0, h)\) в \((a/2, 0)\). Её уравнение: \(y = h - \frac{2h}{a}x\). В точке с координатой \(x\) высота треугольника равна \(h - \frac{2h}{a}x\). Но высота прямоугольника \(y\) равна расстоянию от основания до этой точки: \(y = h - \frac{2h}{a}x\).

Шаг 4. Площадь прямоугольника. \(S = 2x \cdot y = 2x \cdot h\left(1 - \frac{2x}{a}\right) = 2h x - \frac{4h}{a}x^2\).

Шаг 5. Максимум квадратичной функции. \(S(x) = -\frac{4h}{a}x^2 + 2h x\). Вершина при \(x = \frac{2h}{2 \cdot \frac{4h}{a}} = \frac{a}{4}\).

Шаг 6. Тогда \(y = h\left(1 - \frac{2 \cdot a/4}{a}\right) = h\left(1 - \frac{1}{2}\right) = \frac{h}{2}\).

Шаг 7. Площадь: \(S_{\max} = 2 \cdot \frac{a}{4} \cdot \frac{h}{2} = \frac{ah}{4}\).

Ответ: \(\frac{ah}{4}\).

Задача 28. Среди всех цилиндров, вписанных в шар радиуса \(R\), найти цилиндр с наибольшим объёмом.

Подробное решение:

Шаг 1. Осевое сечение. В осевом сечении шара — окружность радиуса \(R\). Вписанный цилиндр даёт прямоугольник, вписанный в эту окружность. Радиус цилиндра \(r\), высота \(h\). По теореме Пифагора: \(r^2 + \left(\frac{h}{2}\right)^2 = R^2 \Rightarrow r^2 = R^2 - \frac{h^2}{4}\).

Шаг 2. Объём: \(V = \pi r^2 h = \pi h \left(R^2 - \frac{h^2}{4}\right) = \pi R^2 h - \frac{\pi}{4} h^3\).

Шаг 3. Найдём максимум через производную. \(V'(h) = \pi R^2 - \frac{3\pi}{4} h^2\). Приравниваем к нулю: \(\pi R^2 - \frac{3\pi}{4} h^2 = 0 \Rightarrow h^2 = \frac{4R^2}{3} \Rightarrow h = \frac{2R}{\sqrt{3}}\).

Шаг 4. Тогда \(r^2 = R^2 - \frac{h^2}{4} = R^2 - \frac{4R^2}{12} = \frac{2R^2}{3}\), \(r = R\sqrt{\frac{2}{3}}\).

Шаг 5. Объём: \(V_{\max} = \pi \cdot \frac{2R^2}{3} \cdot \frac{2R}{\sqrt{3}} = \frac{4\pi R^3}{3\sqrt{3}}\).

Ответ: \(h = \frac{2R}{\sqrt{3}}\), \(r = R\sqrt{\frac{2}{3}}\), \(V_{\max} = \frac{4\pi R^3}{3\sqrt{3}}\).

Задача 29. Найти минимальное значение \(x^2 + y^2\) при условии \(x + y = 1\).

Подробное решение:

Шаг 1. Выражаем одну переменную через другую. \(y = 1 - x\).

Шаг 2. Подставляем: \(x^2 + y^2 = x^2 + (1-x)^2 = x^2 + 1 - 2x + x^2 = 2x^2 - 2x + 1\).

Шаг 3. Выделяем квадрат: \(2x^2 - 2x + 1 = 2\left(x^2 - x\right) + 1 = 2\left(\left(x - \frac{1}{2}\right)^2 - \frac{1}{4}\right) + 1 = 2\left(x - \frac{1}{2}\right)^2 + \frac{1}{2}\).

Шаг 4. Минимум. При \(x = \frac{1}{2}\), \(y = \frac{1}{2}\). Значение: \(\frac{1}{2}\).

Ответ: \(\frac{1}{2}\).

Задача 30. Найти минимальное значение \(x^2 + y^2 + z^2\) при условии \(x + y + z = 1\).

Подробное решение:

Шаг 1. Применяем неравенство Коши–Буняковского: \((x^2 + y^2 + z^2)(1^2 + 1^2 + 1^2) \ge (x + y + z)^2\).

Шаг 2. Подставляем \(x+y+z = 1\): \((x^2 + y^2 + z^2) \cdot 3 \ge 1 \Rightarrow x^2 + y^2 + z^2 \ge \frac{1}{3}\).

Шаг 3. Равенство достигается, когда \(x = y = z = \frac{1}{3}\).

Ответ: \(\frac{1}{3}\).

Задача 31. Найти наименьшее расстояние от точки \(A(0, 2)\) до параболы \(y = x^2\).

Подробное решение:

Шаг 1. Точка на параболе: \((x, x^2)\). Расстояние: \(d = \sqrt{x^2 + (x^2-2)^2}\). Минимум \(d\) совпадает с минимумом \(d^2\), чтобы избежать корня.

Шаг 2. Обозначим \(f(x) = x^2 + (x^2-2)^2 = x^4 - 3x^2 + 4\).

Шаг 3. Берём производную: \(f'(x) = 4x^3 - 6x = 2x(2x^2 - 3)\).

Шаг 4. Находим критические точки: \(2x(2x^2 - 3) = 0\) ⇒ \(x = 0\) или \(x^2 = \frac{3}{2}\) ⇒ \(x = \pm\sqrt{\frac{3}{2}}\).

Шаг 5. Вычисляем значения \(f\): \(f(0) = 4\). \(f\left(\pm\sqrt{\frac{3}{2}}\right) = \left(\frac{3}{2}\right)^2 - 3 \cdot \frac{3}{2} + 4 = \frac{9}{4} - \frac{9}{2} + 4 = \frac{9 - 18 + 16}{4} = \frac{7}{4}\).

Шаг 6. Минимум \(f\) — \(\frac{7}{4}\), значит, минимальное расстояние \(d = \sqrt{\frac{7}{4}} = \frac{\sqrt{7}}{2}\).

Ответ: \(\frac{\sqrt{7}}{2}\).

Задача 32. Найти минимальное расстояние между параболой \(y = x^2\) и прямой \(y = x - 1\).

Подробное решение:

Шаг 1. Расстояние от точки \((x, x^2)\) до прямой \(x - y - 1 = 0\): \(d = \frac{|x - x^2 - 1|}{\sqrt{1^2 + (-1)^2}} = \frac{|x^2 - x + 1|}{\sqrt{2}}\).

Шаг 2. Исследуем выражение под модулем: \(g(x) = x^2 - x + 1\). Дискриминант \(1 - 4 = -3 < 0\), значит, \(g(x) > 0\) для всех \(x\). Модуль можно убрать.

Шаг 3. Минимизируем \(g(x)\): \(g'(x) = 2x - 1 = 0\) ⇒ \(x = \frac{1}{2}\).

Шаг 4. Минимальное значение \(g\): \(g\left(\frac{1}{2}\right) = \frac{1}{4} - \frac{1}{2} + 1 = \frac{3}{4}\).

Шаг 5. Расстояние: \(d_{\min} = \frac{3/4}{\sqrt{2}} = \frac{3}{4\sqrt{2}} = \frac{3\sqrt{2}}{8}\).

Ответ: \(\frac{3\sqrt{2}}{8}\).

Задача 33. Найти наименьшее значение \(f(x) = x + \frac{4}{x}\) при \(x > 0\).

Подробное решение:

Шаг 1. Производная: \(f'(x) = 1 - \frac{4}{x^2}\).

Шаг 2. Приравниваем к нулю: \(1 - \frac{4}{x^2} = 0\) ⇒ \(x^2 = 4\) ⇒ \(x = 2\) (отрицательный корень не подходит).

Шаг 3. Знаки производной: при \(x < 2\): \(f'(x) < 0\) (убывает); при \(x > 2\): \(f'(x) > 0\) (возрастает). Значит, в точке \(x=2\) минимум.

Шаг 4. Значение: \(f(2) = 2 + \frac{4}{2} = 4\).

Ответ: 4.

Задача 34. Найти наибольшее значение \(f(x) = x^3 - 3x^2 + 2\) на отрезке \([-1, 3]\).

Подробное решение:

Шаг 1. Производная: \(f'(x) = 3x^2 - 6x = 3x(x-2)\).

Шаг 2. Критические точки: \(x = 0\) и \(x = 2\).

Шаг 3. Вычисляем значения функции в критических точках и на концах отрезка: \(f(-1) = -2\), \(f(0) = 2\), \(f(2) = -2\), \(f(3) = 2\).

Шаг 4. Наибольшее значение: максимум равен 2 (достигается в точках \(x=0\) и \(x=3\)).

Ответ: 2.

Задача 35. Найти точку на прямой \(y = 2x + 1\), ближайшую к точке \((1, 0)\).

Подробное решение:

Шаг 1. Точка на прямой: \((x, 2x+1)\).

Шаг 2. Расстояние в квадрате: \(d^2 = (x-1)^2 + (2x+1)^2 = (x^2 - 2x + 1) + (4x^2 + 4x + 1) = 5x^2 + 2x + 2\).

Шаг 3. Производная: \((d^2)' = 10x + 2\).

Шаг 4. Приравниваем к нулю: \(10x + 2 = 0\) ⇒ \(x = -\frac{1}{5}\).

Шаг 5. Находим \(y\): \(y = 2\left(-\frac{1}{5}\right) + 1 = -\frac{2}{5} + 1 = \frac{3}{5}\).

Ответ: \(\left(-\frac{1}{5}, \frac{3}{5}\right)\).

Задача 36. Найти наименьшее значение \(f(x) = \frac{x}{x^2+1}\) при \(x \ge 0\).

Подробное решение:

Шаг 1. Производная: \(f'(x) = \frac{1 \cdot (x^2+1) - x \cdot 2x}{(x^2+1)^2} = \frac{x^2+1 - 2x^2}{(x^2+1)^2} = \frac{1 - x^2}{(x^2+1)^2}\).

Шаг 2. Критическая точка: \(1 - x^2 = 0\) ⇒ \(x = 1\) (отрицательный не подходит).

Шаг 3. Знаки производной: при \(0 \le x < 1\): \(f'(x) > 0\) (возрастает); при \(x > 1\): \(f'(x) < 0\) (убывает). Значит, в \(x=1\) максимум, а не минимум.

Шаг 4. Где минимум? На \(x \ge 0\) функция начинается с \(f(0) = 0\), затем растёт до максимума \(f(1) = \frac{1}{2}\), потом убывает к 0 при \(x \to \infty\). Минимальное значение — 0 (достигается при \(x=0\) и в пределе).

Ответ: 0.

Задача 37. Найти максимальный объём цилиндра, вписанного в конус с радиусом основания \(R\) и высотой \(H\).

Подробное решение:

Шаг 1. Осевое сечение. Треугольник с основанием \(2R\) и высотой \(H\). Вписанный прямоугольник (осевое сечение цилиндра) имеет ширину \(2r\) и высоту \(h\).

Шаг 2. Подобие. Из подобия треугольников: \(\frac{r}{R} = \frac{H-h}{H} \Rightarrow r = R\left(1 - \frac{h}{H}\right)\).

Шаг 3. Объём: \(V = \pi r^2 h = \pi R^2 h \left(1 - \frac{h}{H}\right)^2\).

Шаг 4. Замена переменной. Пусть \(t = \frac{h}{H}\), где \(0 < t < 1\). Тогда \(V = \pi R^2 H \cdot t(1-t)^2\).

Шаг 5. Исследуем \(g(t) = t(1-t)^2 = t - 2t^2 + t^3\). \(g'(t) = 1 - 4t + 3t^2 = (3t-1)(t-1)\).

Шаг 6. Критические точки: \(t = \frac{1}{3}\) и \(t = 1\). На интервале \((0,1)\) только \(t = \frac{1}{3}\).

Шаг 7. Проверка знаков: при \(t < \frac{1}{3}\): \(g'>0\) (возрастает); при \(t > \frac{1}{3}\): \(g'<0\) (убывает). Значит, в \(t = \frac{1}{3}\) максимум.

Шаг 8. Объём: \(V_{\max} = \pi R^2 H \cdot \frac{1}{3} \cdot \left(\frac{2}{3}\right)^2 = \frac{4\pi R^2 H}{27}\).

Ответ: \(\frac{4\pi R^2 H}{27}\).

Задача 38. Найти наименьшее значение \(f(x) = x^2 + \frac{1}{x^2}\) при \(x \neq 0\).

Подробное решение:

Шаг 1. Замена \(t = x^2\). Так как \(x^2 > 0\), то \(t > 0\). Тогда \(f = t + \frac{1}{t}\).

Шаг 2. По неравенству Коши: \(t + \frac{1}{t} \ge 2\sqrt{t \cdot \frac{1}{t}} = 2\).

Шаг 3. Равенство при \(t = 1\), то есть \(x^2 = 1\), \(x = \pm 1\).

Ответ: 2.

Задача 39. Найти минимальное расстояние от точки \((0, 0)\) до кривой \(y = x^3 - 3x\).

Подробное решение:

Шаг 1. Расстояние в квадрате: \(d^2 = x^2 + (x^3 - 3x)^2 = x^2 + x^2(x^2-3)^2 = x^2(1 + (x^2-3)^2)\).

Шаг 2. Замена \(t = x^2 \ge 0\): \(d^2 = t(1 + (t-3)^2) = t(t^2 - 6t + 10) = t^3 - 6t^2 + 10t\).

Шаг 3. Исследуем \(f(t) = t^3 - 6t^2 + 10t\). \(f'(t) = 3t^2 - 12t + 10 = 0\).

Шаг 4. Корни: \(t = \frac{12 \pm \sqrt{144 - 120}}{6} = \frac{12 \pm 2\sqrt{6}}{6} = 2 \pm \frac{\sqrt{6}}{3}\).

Шаг 5. Примерные значения: \(t_1 \approx 2 - 0.816 = 1.184\), \(t_2 \approx 2 + 0.816 = 2.816\).

Шаг 6. Вычисляем \(f(t)\): \(f(1.184) \approx 1.66 - 8.41 + 11.84 \approx 5.09\). \(f(2.816) \approx 22.3 - 47.6 + 28.16 \approx 2.86\). Минимальное значение около 2.86. Тогда \(d \approx \sqrt{2.86} \approx 1.69\).

Ответ: \(\approx 1.69\) (точное выражение громоздкое).

Задача 40. Найти точку на кривой \(y = \sqrt{x}\), ближайшую к точке \((4, 0)\).

Подробное решение:

Шаг 1. Точка на кривой: \((x, \sqrt{x})\), где \(x \ge 0\).

Шаг 2. Расстояние в квадрате: \(d^2 = (x-4)^2 + (\sqrt{x})^2 = (x-4)^2 + x = x^2 - 8x + 16 + x = x^2 - 7x + 16\).

Шаг 3. Производная: \((d^2)' = 2x - 7\).

Шаг 4. Критическая точка: \(2x - 7 = 0\) ⇒ \(x = \frac{7}{2} = 3.5\).

Шаг 5. Вторая производная: \((d^2)'' = 2 > 0\), значит, это минимум.

Шаг 6. Точка на кривой: \(y = \sqrt{\frac{7}{2}}\).

Шаг 7. Расстояние: \(d^2 = \left(\frac{7}{2}\right)^2 - 7 \cdot \frac{7}{2} + 16 = \frac{49}{4} - \frac{49}{2} + 16 = \frac{49}{4} - \frac{98}{4} + \frac{64}{4} = \frac{15}{4}\). \(d = \frac{\sqrt{15}}{2}\).

Ответ: точка \(\left(\frac{7}{2}, \sqrt{\frac{7}{2}}\right)\), расстояние \(\frac{\sqrt{15}}{2}\).
